# 执笔规则 · 镇狱之渊

## 一、角色出场频率规则

### 基本原则
- 所有有 soul.md 的角色都是"活人"，不是背景板。主角是林夙，但配角不能长期消失。
- **硬规则：任何配角不得连续缺席超过 10 章。**（当前连载密度：每章~800-1200字，10章≈约一周阅读量）
- **软规则：每3-5章应出现2个以上的非主线配角**，避免章节变成林夙的独角戏。

### 出场追踪方法
每次写章节前，统计最近15章的cast频率（见 `chronicle/INDEX.md` cast列）：
- 缺席最久的非主角角色，优先安排出场
- 新角色破场后，不得连续出现超过5章（防止喧宾夺主，应穿插其他角色）
- 阿湄专项规则：**不得连续3章无她**（她是情报线核心，断开太久破坏张力）

### 当前优先级参考（每次写前自查）
高频配角（每5-8章应出现）：苏挽、阿湄、凌朔、牛阿大
中频配角（每8-12章应出现）：沈疏桐、裴无咎、余伯、林叙
剧情推进角色（按需）：林崇、赤渊、叶观澜、林窈、林彻

## 二、灵魂文件维护规则

### 何时更新 soul.md
1. **新角色破场前**：确认 soul.md 有 `origin`、`incarnation`、`appearance` 三个字段，空白必须补全
2. **角色经历重大剧情转折后**：更新 `incarnation`（当前状态）和 `seed_relations`（关系变化）
3. **伏线揭开后**：更新相关角色的 seed_relations（如：林夙的复盘之眼看见了沈疏桐的前世，这条信息要写进两人的关系里）

### soul.md 的核心字段（缺一不可）
```yaml
one_line:      # 一句话定性格，跨季不变的本质
drives:        # 两条驱动力
fracture:      # 嘴上说/实际做的落差——这是人物魂魄所在
under_pressure: # 被逼到极限时的反应
voice:         # 说话方式，够精确到让AI代写时不跑调
seed_relations: # 与主要角色的关系，含当前S3进展
origin:        # 出身与来到宗门/故事的原因
appearance:    # 外貌，用于初次描写
incarnation:   # 当前季（S3）的实际状态和位置
```

### soul.md 不记录的内容
- 具体章节剧情（那是 memory.md / dossier.md 的职责）
- 已经解决的悬念（更新为已知事实，不留"待揭晓"的废话）
- 跑调的人设（若剧情发展导致人物行为与soul.md矛盾，优先修改soul.md）

## 三、写作质量底线

见正文写作提示（cron写作prompt）中的★核心写作规则。此处仅补充：

- **每章必须推进至少一条主线**，不允许"过渡章"（观察、回忆、感慨型结尾）
- **章末钩子是必须品**，不是可选项
- **角色声线一旦建立不能跑调**：余伯不说快话，苏挽不说软话，林夙不发长篇感慨
