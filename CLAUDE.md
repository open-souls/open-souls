# 执笔规则 · 镇狱之渊

---

## 一、章节标题规则

**标题长度：2-4字，禁止纯单字。**

单字标题（《迹》《灶》《债》）太薄，无法传达场景重量。

好标题标准：
- 2字：双重含义或场景核心词（《归体》《白火》《旧账》《换命》）
- 3字：动宾或修饰结构（《还债人》《监守令》《空格子》）
- 4字：四字成语感，文言节奏（《监守自盗》《还债入局》）
- 避免：纯地点（《后院》）、纯人名（《余伯》）、语气词

---

## 二、角色出场规则

### 硬规则：每章 cast 最少 4 人，主线章最好 5-6 人

主角林夙常驻，配角至少 3 人起。理由：
- 单次见面只有 2-3 人的章节节奏偏私小说，仙侠连载需要群像感
- 多人场景自然产生声线碰撞，信息量更高，爽点密度更容易维持
- 读者需要定期看见自己喜欢的角色，否则追感断裂

**例外允许**：1对1场景（如林夙与赤渊正面对峙）可以 3 人，但这种章节前后必须是群像章。

### 缺席上限（硬规则）
- 任何配角不得连续缺席超过 **10 章**
- 阿湄专项：不得连续 **3 章** 无她
- 新角色破场后，不得连续超过 **5 章**（防止喧宾夺主）

### 每章写前：必须统计最近 15 章 cast 频率
从 `chronicle/INDEX.md` cast 列数，缺席最久的角色优先安排。

### 出场优先级
| 频率 | 角色 | 目标 |
|------|------|------|
| 高频（每5-8章） | 苏挽、阿湄、凌朔、牛阿大 | 不能长期缺席 |
| 中频（每8-12章） | 沈疏桐、裴无咎、余伯、林叙 | 定期出现 |
| 剧情触发 | 林崇、赤渊、叶观澜、林窈、林彻、云栀 | 按需 |

## 三、群像写法规则

每个出场角色，必须有至少一件只属于自己的事：
- 一个独有的小动作（裴无咎嚼包子、牛阿大把粥推过去）
- 一句体现声线的台词（不能和主角口吻雷同）
- 一个只有他/她才会有的情绪反应

**禁止工具人式出场**：角色不是为了给林夙递消息、说信息点而存在，是这个人活在这个场景里。

## 四、灵魂文件维护规则

### 何时更新 soul.md
1. **新角色破场前**：确认 soul.md 有 `origin`、`incarnation`、`appearance` 三个字段
2. **角色经历重大剧情转折后**：更新 `incarnation`、`seed_relations`
3. **伏线揭开后**：更新相关角色的 seed_relations

### soul.md 核心字段（缺一不可）
```yaml
one_line:       # 一句话定性格，跨季不变
drives:         # 两条驱动力
fracture:       # 嘴上说/实际做的落差
under_pressure: # 极限时的反应
voice:          # 说话方式
seed_relations: # 与主要角色关系，含S3进展
origin:         # 出身与来宗门的原因
appearance:     # 外貌
incarnation:    # 当前S3实际状态
```

## 五、写作质量底线

- **每章推进至少一条主线**，不允许纯过渡章
- **章末钩子必须品**，最后一句话要有重量
- **声线不能跑调**：余伯不说快话，苏挽不说软话，裴无咎不说沉话
- **hook 字段含引号时，必须用 block scalar（|）格式**，否则 YAML 解析报错

## 六、frontmatter 标准格式

```yaml
---
season: 3
chapter: N
title: 2-4字标题
cast: [林夙, 角色B, 角色C, 角色D, 角色E]  # 最少4人
pov: 林夙
beat: 转·宗门内部暗流，轮回秘密险些全揭 序号/总数
ships:
  林夙×苏挽: 本章嗑点（若有）
hook: 章末钩子（含引号时用 | 格式）
---
```
