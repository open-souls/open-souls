# 执笔规则 · 镇狱之渊

---

## 零、叙事结构 · 多线并行 × 男女频平衡（GoT 式）

> **这是全书最高层规则，覆盖以下所有条目。** 本书不是「林夙单主线 + 配角围着他转」，是**像权力的游戏一样的多线群像**：几条故事线并行，各有自己的 POV、赌注、爽点/痛点，彼此交织。林夙线最重，但**不是唯一主线，也不独占视角**。

### 1. 男频线 / 女频线（两套看点，必须都喂饱）

| | 男频线（格局/权力） | 女频线（情感/关系） |
|---|---|---|
| 看点 | 复仇、升级、打脸、权谋布局、阵营博弈、势力战争、实力碾压、江湖格局 | 拉扯、克制、虐心、关系升温、人物内心、命运纠葛 |
| 典型线 | 林夙复仇升级线、宗门夺嫡/权谋线、叶观澜布局线、江湖势力线 | 三角（林夙×苏挽×阿湄）、苏挽弧、阿湄弧、叶观澜×阿湄前史 |
| POV 锚 | 林夙 / 凌朔 / 裴无咎 / 叶观澜（反派布局视角） | 苏挽 / 阿湄 / 林夙（关系中的他） |
| 爽点节奏 | 三章一小爆（小胜/识破），十章一大爆（翻盘/碾压） | 三章一锚点，每条线按 §七 升温曲线 |

**核心：男频要「爽」，女频要「虐」。同一本书两种读者都要追下去。**

### 2. 平衡配额（硬规则）

- **每 10 章**：男频主导 ≥3 章、女频主导 ≥3 章、混合 ≤4 章。
- **不得连续 5 章纯女频，也不得连续 5 章纯男频。** 跑偏即回炉。
- 单章可以是混合，但 frontmatter `line` 字段要标清本章**主导**是哪频（见 §六）。

### 3. 多 POV 轮换（GoT 式）

- **POV 不再默认林夙。** POV 角色池 = {林夙、苏挽、阿湄、裴无咎、叶观澜、凌朔…}，每条线由它的 POV 角色扛。
- **不得连续 6 章同一 POV。** 切 POV 本身就是节奏工具。
- 切 POV 时严守 §七维 6「只写该 POV 能感知到的，判断都带他的色」——换人就换那双眼睛和那套判断。
- 反派 POV（叶观澜布局视角）允许且鼓励：让读者先于角色看到局，制造「细思极恐」信息差。

### 4. 交织规则（线不能各跑各的）

- 每条线必须在某节点与**另一条线交叉**：一个事件同时推进两条线（如一场宗门变故，既是林夙的权谋翻盘=男频，也是苏挽越线的触发=女频）。
- **切走式钩子**：这一章把 A 线推到高点，下一章切 B 线吊着——多线叙事最有力的章末钩子（见 §五）。
- 任一条线不得连续缺席超过 **8 章**（防读者掉线）。

### 5. 雅俗共赏 · 双层文本（另一根同级的轴）

男女频是「喂哪种看点」，雅俗是「喂哪种深浅」——**两根轴正交，都要全覆盖**。同一本书要让**只要爽的空心读者**追更，也让**会读文学的高级读者**读出回甘：
- **表层（爽）是地板，必达**：钩子/爽点/打脸/期待感——空心读者只读这一层也爽完全程。
- **深层（文学）是叠加红利，不伤表层**：主题/隐喻/潜台词/命运反讽——读不到的人剧情照样爽，读到的人多赚一层。
- **一笔写两层**：每个关键句既给空心读者爽点，又给高级读者多一层（范文H 秤砣就是范本）。
- **被欣赏最大化**：表层保下限（人人有爽）× 深层抬上限（高级读者多赚），且深层绝不写进表层的承重墙。

> 细则、双重编码技法、上线增检 3 条见 `docs/standards/雅俗共赏.md`。

### 6. 不变的底线

林夙仍是全书引力最重的人物、情感三角的焦点；GoT 化指的是**结构与视角**，不是把他降级成路人。**男频升级线与女频情感线在他身上交汇**——这是本书区别于纯男频/纯女频的根。

## 一、章节标题规则

**标题长度：2-4字，禁止纯单字。**

单字标题（《迹》《灶》《债》）太薄，无法传达场景重量。

好标题标准：
- 2字：双重含义或场景核心词（《归体》《白火》《旧账》《换命》）
- 3字：动宾或修饰结构（《还债人》《监守令》《空格子》）
- 4字：四字成语感，文言节奏（《监守自盗》《还债入局》）
- 避免：纯地点（《后院》）、纯人名（《余伯》）、语气词

## 二、角色出场规则

### cast 人数：按场景决定，但必须有节奏变化

没有固定下限——3 人可以是最佳，1 人独角戏也可以。**真正的问题是连续同一规模**。

| 场景类型 | 建议人数 | 典型用法 |
|---------|---------|---------|
| 正面对峙 / 审问 | 2-3 人 | 林夙×赤渊、林夙×余伯 |
| 情报会合 / 群议 | 4-6 人 | 全员聚集、出发前部署 |
| 独角内省 / 行动 | 1 人 | 林夙夜间潜入、独自读账 |
| 群像日常 | 5+ 人 | 宗门集体场景、节日贺宴 |

**硬规则：不得连续 4 章 cast 数量相同。** 节奏必须有大有小，读者才不会麻木。

### 缺席上限（硬规则）
- 任何配角不得连续缺席超过 **10 章**
- 阿湄专项：不得连续 **3 章** 无她
- 新角色破场后，不得连续超过 **5 章**（防止喧宾夺主）

### 每章写前：必须统计最近 15 章 cast 频率
从 `chronicle/INDEX.md` cast 列数，缺席最久的角色优先安排。

### 出场优先级
| 频率 | 角色 | 目标 |
|------|------|------|
| 高频（每5-8章） | 苏挽、阿湄、凌朔、牛阿大 | 不能长期缺席 |
| 中频（每8-12章） | 沈疏桐、裴无咎、余伯、林叙 | 定期出现 |
| 剧情触发 | 林崇、赤渊、叶观澜、林窈、林彻、云栀 | 按需 |

### POV 角色池（与 §零-3 配套）
扛线的 POV 角色，轮换使用，不得连续 6 章同一 POV：
| POV 角色 | 主扛哪条线 | 频 |
|---------|-----------|---|
| 林夙 | 复仇升级线（男频）/ 情感三角（女频） | 最高 |
| 苏挽 | 情感线 + 轻度黑化线（女频）/ 宗门权谋（男频） | 高 |
| 阿湄 | 阿湄弧 + 叶观澜前史（女频）/ 江湖杀伐（男频） | 高 |
| 裴无咎 / 凌朔 | 江湖势力 · 权谋线（男频） | 中 |
| 叶观澜 | 反派布局线（男频，信息差） | 剧情触发 |

## 三、群像写法规则

### 总则：每个角色都是活生生的、独立的人

**这是不可让步的红线，凌驾于剧情便利之上。** 书里没有「配角」，只有暂时不在 POV 里的人。每个角色由**他自己的人设**驱动，有自己的生活、目标、心理戏——他存在，不是因为主角需要他，是因为他本来就在那儿过自己的日子。

**每个人都是真的人 → 每个人都要有人物小传（硬规则）。** 任何角色破场前，必须在 `souls/角色名/soul.md` 的 **`## Dossier` 段**写好他的**人物小传**——基础档案 / 出身与来处 / 外貌锚点 / 性格底色 / 驱动力 / 裂缝 / 极限反应 / 声线指纹 / 关键关系 / 当前状态 / 标志性细节 / 写作禁区 / 备忘。格式照已完成范例（`souls/凌朔/soul.md`、`souls/沈疏桐/soul.md`、`souls/裴无咎/soul.md`）与 `souls/_TEMPLATE/soul.md` 末尾骨架。**没有 `## Dossier` 小传的人不许破场登台**：没有底子的角色，写出来必然是工具人。
> 区分：`soul.md`（魂 + `## Dossier` 人物小传，手写）是权威；`souls/角色名/dossier.md` 与 `CAST.md` 是 **`engine/trace.py` 自动生成的索引**（关系数值/记得的事/出场），**勿手写**——改完 soul/state 跑 `python engine/trace.py` 刷新。

判据（每个出场角色都要过）：
- **有自己的人设逻辑**：他做的事、说的话，是从他的 `soul.md`（drives / fracture / under_pressure / voice）长出来的，换个角色就不成立。
- **有自己的生活与目标**：镜头外他也有要忙的事、要怕的事、要图的事——哪怕这一章只露一句，也能让读者感到「他不只为这场戏活着」。
- **有自己的心理戏**：他在这一幕里有自己的盘算/情绪/挣扎，且**不必都说出来**——可以只用一个动作、一个停顿、一处目光泄露（见 §七维 4/6）。
- **独立于主角**：把林夙抽走，这个人依然有他自己的处境和动机；他对林夙的态度是他的选择，不是剧情指派。

### 每个出场角色，必须有至少一件只属于自己的事
- 一个独有的小动作（裴无咎嚼包子、牛阿大把粥推过去）
- 一句体现声线的台词（不能和主角口吻雷同）
- 一个只有他/她才会有的情绪反应

### 禁止工具人式出场（硬规则）
**角色不是为了给林夙递消息、说信息点、推进主角剧情而存在。** 命中以下任一即工具化，回炉：
- 这个角色这一章**只**做了「传话 / 解释设定 / 给主角搭台」，没有一丝属于他自己的东西；
- 他的台词换成任何人说都成立（没有声线指纹）；
- 他的情绪/选择只服务于主角的爽点或虐点，自己不承担后果、不留下余味；
- 他被叫来又被忘掉——出场后既无目标推进，也无心理痕迹。

> 验收一问：**如果这一幕的 POV 切到他身上，他这一章过得是不是一段完整的、有他自己分量的日子？** 不是，就还没写活。

## 四、灵魂文件维护规则

每个角色，`soul.md` 一份文件两层内容（见 §三 红线）：
- **frontmatter + 底色** — 魂：一句话定性、drives、fracture、声线。跨季不变（皮囊会换，魂不变）。
- **`## Dossier` 段** — 人物小传：基础档案 / 出身与来处 / 外貌锚点 / 性格底色 / 驱动力 / 裂缝 / 极限反应 / 声线指纹 / 关键关系 / 当前状态 / 标志性细节 / 写作禁区 / 备忘。**这一世这个人的全部底子，手写，权威。**

> `souls/角色名/dossier.md` 与 `CAST.md` 是 `engine/trace.py` 从 soul + state + 出场**自动生成的索引**，不是小传本体，**勿手写**。

### 何时建 / 更新 `## Dossier`（人物小传）
1. **任何角色破场前（硬规则）**：soul.md 必须已有 `## Dossier` 段，否则不许登台。
2. **角色经历重大剧情转折后**：更新「当前状态」「关键关系」。
3. **伏线揭开后**：更新「出身与来处」「关键关系」。
4. **改完后**跑 `python engine/trace.py` 刷新 `dossier.md` + `CAST.md` 索引。

### 何时更新 soul.md
1. **新角色破场前**：确认 soul.md 有 `origin`、`incarnation`、`appearance` 三个字段
2. **角色经历重大剧情转折后**：更新 `incarnation`、`seed_relations`
3. **伏线揭开后**：更新相关角色的 seed_relations

### soul.md 核心字段（缺一不可）
```yaml
one_line:       # 一句话定性格，跨季不变
drives:         # 两条驱动力
fracture:       # 嘴上说/实际做的落差
under_pressure: # 极限时的反应
voice:          # 说话方式
seed_relations: # 与主要角色关系，含S3进展
origin:         # 出身与来宗门的原因
appearance:     # 外貌
incarnation:    # 当前S3实际状态
```

## 五、写作质量底线

- **每章推进至少一条主线**，不允许纯过渡章
- **每章标清主导频**（男频/女频/混合），整体守 §零-2 配额；连续 5 章同频即跑偏，回炉
- **章末钩子必须品**，最后一句话要有重量；多线时优先用「切走式钩子」（A 线推到高点→下一章切 B 线）
- **声线不能跑调**：余伯不说快话，苏挽不说软话，裴无咎不说沉话
- **切 POV 即切那双眼睛**：换 POV 角色后，判断、用词、能感知的范围全部随之换（§七维 6）
- **hook 字段含引号时，必须用 block scalar（|）格式**，否则 YAML 解析报错

## 七、情感与亲密场景系统

> **定位（与 §零 配套）**：§七–§九 是**女频线的骨架**。它们与男频线（复仇升级 / 宗门权谋 / 江湖势力 / 叶观澜布局）**并行**，不是全书唯一内容——按 §零-2 配额穿插，别让情感线连吃 5 章把男频读者熬走。

### 总原则

**亲密场景不是独立章节，是被前面十几章喂熟的东西。**
到它真的发生时，读者应该感到的是"终于"，不是"突然"。

史官有权在任何章节埋伏笔，也有权**回改已完成章节**以为后文铺路。
回改原则：**只加，不减情节**。改的是细节层——一个动作、一个感官描述、一行目光——不动对话主干和剧情走向。

---

### 伏笔五种类型

#### 1. 感官锚点（最重要）
绑定一个具体的身体细节，第一次出现是中性的，后续每次出现都带上积累的重量。

**已有锚点（勿重复，要呼应）：**
- 林夙×阿湄：她"推碗"的手（ch123）；"就是想看看你"的眼神（ch126）
- 林夙×苏挽：她"坐在旁边，陪一段安静"的距离；她手放在桌上的方式

**新锚点埋法：**
- 选择一个不起眼的身体部位（手腕、颈侧、指节、耳根），让它第一次出现在毫无情绪的语境里
- 第二次出现时，语境稍有不同——角色注意到了，但没说
- 第三次以后，那个部位变成了一个"开关"

#### 2. 距离记录
每次两人异常靠近，必须写清楚那个距离——不是"靠近了"，是"他能听见她呼吸换了节奏"或"她的袖角在他手边两寸"。  
距离在不同章节里必须有变化：第一次是六尺，第二次是三尺，最后是没有距离。

#### 3. 未竟的停顿
两人之间某件事差点发生，但被打断（有人进来、某人转开话题、某人主动走了）。  
这个停顿**必须写得很具体**——停在哪里，两人各自的状态，然后什么打断了它。  
打断之后，必须有一方"记住了"——哪怕只是"他把那件事往下压了一下"。  
**这个未竟的停顿，是后文真正发生时的直接引线。**

#### 4. 声音温度变化
角色说话声音在特定场合会变——低一点，慢一点，或者反常地轻。  
第一次出现：不解释，就是写出来。  
后续：只要那个场合重现（夜晚、两人独处、情绪下沉），就触发同一种声音描述，形成条件反射。

#### 5. 重复的小动作
一个角色习惯性地对另一个人做某件小事（替他整领子、帮她拨开眼前的头发、把东西塞进他手里）。  
第一次：日常，无情绪标记。  
后来：在某个情绪高点，同样的动作出现——它变成了告白，不需要台词。

---

### 跨章针线规则

**埋线时**：在 frontmatter 的 `ships` 字段注明这一章埋了什么伏笔，供后续章节查阅。  
例：`林夙×阿湄: 【锚点·手】她接过茶罐时，他看了她的手一眼，没说话`

**引线时**：在正文里直接呼应，不需要解释——读者自己会想起来。  
例：之后某章，他的目光落在她手上，只写"他看了她的手，想起什么，没说"——读者填空。

**同时维护三条主线伏笔**：林夙×苏挽、林夙×阿湄、苏挽×阿湄（三条线相互影响，不独立运作）。

### 升温节奏（硬性规定，不得跳级）

```
第0级：感官锚点建立（无情绪标记的身体细节）
第1级：距离缩短 + 未竟停顿（差点，被打断）
第2级：主动的小动作（有一方主动做了一件事，不多解释）
第3级：第一次真实的身体接触（不是意外，是选择，但仍然克制）
第4级：暧昧对峙（两人都知道，谁也不先说）
第5级：临界（一个人或两个人都越了线，但还没到）
第6级：发生
```

每两级之间，至少间隔 3 章其他内容（剧情、别的角色）。
不许一章连跳两级。

### 亲密场景文法（第6级到达时）

**写法原则：**
- 文言感，短句切镜，不写生理术语
- 感官先于动作：温度、重量、呼吸、光线——这些比动作本身更重要
- 每个动作都是选择，不是冲动——写出角色在做决定的那个停顿
- 高潮前必须有一句对话或一个眼神，否则读者不知道这是两个人的事

**禁止写法：**
- "他/她一下子就……"（跳戏）
- 动作列表（A动作、B动作、C动作——像说明书）
- 突然的体力描写（先建立情绪，再建立身体）
- 结束后若无其事（必须有后续重量，哪怕只是一句话）

**林夙专属写法**：他不会主动开口，他的"主动"是不走、不挡、不转开眼神。  
**阿湄专属写法**：她会说话，但那句话会说得像是在说别的事。  
**苏挽专属写法**：她会先做，后来才想清楚自己为什么做了。

## 八、三角关系系统（林夙 × 苏挽 × 阿湄）

### 性别不设限

三角关系内，所有两两组合均可发展亲密情节：
- 林夙 × 苏挽（正线）
- 林夙 × 阿湄（正线）
- **苏挽 × 阿湄（独立成立，不依附林夙）**

苏挽×阿湄 不是"争同一个人"的镜像，是两条独立的情感线在某个点相遇，产生了只属于她们自己的东西。

---

### 好感度基准（S3当前，随剧情更新）

| 方向 | 好感度 | 质地 |
|------|--------|------|
| 林夙 → 苏挽 | ████░ 深 | 最久，最可信，还没被命名 |
| 林夙 → 阿湄 | ███░░ 中深 | 说不清是什么，但她在他会留意的地方 |
| 苏挽 → 林夙 | ████░ 深 | 她在意，但她的"在意"比她说的，重很多 |
| 苏挽 → 阿湄 | ██░░░ 中 | 戒备 + 认可 + 某种说不清的注意 |
| 阿湄 → 林夙 | ████░ 深 | 知道得太多，反而没法轻易 |
| 阿湄 → 苏挽 | ██░░░ 中 | 已读懂她，但还没决定用这份懂做什么 |

**更新规则**：每次出现苏挽×阿湄场景后，在 ships 字段标注当前好感度变化。

---

### 苏挽 × 阿湄 的人物化学

**为什么她们会相互吸引（不是因为林夙）：**
- 阿湄的"油滑"对苏挽不起效——苏挽不被逗，不被惹，只是稳稳地看着她。阿湄一生里遇见过几个看不穿的人，苏挽是其中之一，她对这种人有天然的注意。
- 苏挽感知到阿湄在说话里藏了很多东西，但从不多问。这种"不追"本身，让阿湄觉得她和别人不同。
- 两个人都在用不同的方式，爱着同一个复杂的人——这件事，把她们放在同一个坐标系里，比任何共同话题都更接近。

**她们的动态节奏（三种模式）：**

#### 模式A：好姐妹（好感度双方均≥中）
- 共同关心同一件事（通常是林夙或当前剧情危机）
- 阿湄用说话挤兑她，苏挽回得比阿湄快半拍——阿湄反而高兴
- 肩并肩做一件事，不需要解释彼此为什么在
- 暖场用，节奏调剂用，为之后更深的场景做地基

#### 模式B：暗流（好感度出现"不对等"或某方有情绪）
- 一方做了什么，另一方注意到了，但不说
- 阿湄的眼神在某一刻，落在苏挽身上的方式，比"朋友"重一点
- 苏挽某次帮了阿湄一件她没开口要的事——而后者停了一下，没有谢，只是看了她一眼
- 用于建立苏挽×阿湄的锚点

#### 模式C：吃醋争宠（林夙突然来访，或三人同场）
- 两人都在，林夙进来——本来是独立的两段关系，现在在同一个空间叠加
- **吃醋写法**：不写"我在意"，写行为变化——阿湄话突然多了，苏挽突然话少了，或者反过来
- **争宠写法**：两人各自做了一件事，方向都是林夙，然后互相看了一眼，都知道刚才自己在做什么
- **林夙写法**：他不是猎手，是一个有点手足无措、但不讨厌这种手足无措的人。他的反应，是读者的镜子。
- **收尾写法**：必须有一个人先笑，或先说一句不相关的话，打破那个空气——通常是阿湄

---

### 三人同场的叙事规则

**林夙突然来访（发现她们在一起）：**
- 门开之前：写清楚她们当时在干什么、距离多近、气氛是什么
- 门开之后：写林夙看到的那个画面（而不是立刻切入对话）
- 他停了多久——一息还是两息——这个停顿很重要
- 如果有暧昧感：他不一定说出来，但他的后续行为会变一点

**三人同场的对话节奏：**
- 阿湄主导节奏（她话多，会活跃气氛）
- 苏挽是锚（她的一句话，会让整个空气停一下）
- 林夙是焦点（但他自己不知道）
- 不允许三人一起说"大道理"——必须有人在做别的事（吃东西、整理、往外看）

**禁止写法：**
- 三人之间突然出现"我要告白"的紧张感（节奏太跳）
- 苏挽和阿湄正面冲突（她们都太聪明，不会正面冲突）
- 林夙充当裁判（他没有资格，他自己也是当事人）

## 九、角色长弧设计（跨季级别）

### 苏挽弧：退婚之后，慢慢的，不自觉的爱；林夙"死"后，慢慢的，越过自己划的线

**苏挽弧分两段，中间以林夙假死为分界。**

---

#### 第一段：行为先于意识（当前进行中）

**核心逻辑：行为先于意识。**
她不会在某章"发现自己喜欢他"——她只是一直在做一些事，有一天回头，发现那些事都是为了他。

**埋伏笔规则：**
- 她做了某件她本来不会做的事→不解释→读者看出来
- 她说了某句话，说完，自己停了一下，像是刚才那句话，比她以为的，重了一点
- 她先到了某个地方，林夙来了，她没有走——而她本来会走
- 某次他被人伤了，她的反应，比她自己预期的，快了一拍

**禁止写法：**
- 她主动开口说"我喜欢你"（她不会）
- 她突然软下来（她的软，是静的，不是融的）
- 她用嫉妒来表达在意（她会用沉默）

**苏挽×林夙 已有锚点：**
- ch124：她坐他旁边，不是对面（距离锚点）
- ch127：她用"你父亲"而不是"林崇"（她懂他那道伤在哪里）
- ch129：阿湄说了什么，她说"没什么"——那个"没什么"里有一件她在保护的事

---

#### 第二段：苏挽轻度黑化（林夙假死之后）

**核心逻辑：她越来越不相信规则能保护她在乎的人，所以她开始自己来。**

林夙死之前，苏挽是那种"在规则边界上行走，但不越过"的人。林夙死之后，她发现她在乎的人已经没了——规则没有保住他。那个"规则能保护我在乎的人"的信念，碎了。

她的黑化，不是戏剧性的转变，是一系列小的"我来处理"的积累：
- 第一次：她没有上报某个消息，自己留着
- 第二次：她用了一种她以前不会用的方式，逼出一个人的实话
- 第三次：她做了一件事，事后，她知道她如果没有遇见阿湄，她不会做
- 她不后悔。这才是真正的轻度黑化——不是痛苦地越界，是平静地越界，然后发现越线之后，她能站稳

**苏挽×阿湄 双人复仇弧：**

这段时间，苏挽和阿湄是搭档——不是因为她们选择了彼此，是因为她们都指向同一件事，而且，她们俩加在一起，才能做完那件事。

- **阿湄是行动者**：她知道怎么做，做起来不眨眼，知道那个世界的规则
- **苏挽是判断者**：她来决定打哪个、放哪个、什么时候停
- **但苏挽的判断越来越带着阿湄的影响**：她的"停"的阈值，比以前，高了；她的"这个不能做"变成了"这个，目前还不用做"

两个人之间的化学：
- 阿湄第一次看见苏挽做了一件"不像她"的事，会有一个停顿——那个停顿，不是评判，是某种她自己也说不清的东西
- 苏挽做了那件事之后，不解释，不道歉——阿湄对这种"做了不说"的人，会莫名地安静一下
- 她们各自都知道：另一个人，看见了她的某个平时藏着的东西

**苏挽黑化的底色：有他的影子**
她做的那些越线的事，细看，都是在替他做他会想做、但来不及做的事。她以为她是在复仇，但实际上，那更像是她用另一种方式，继续做了他们一起开始的那件事。

**苏挽禁止写法（第二段）：**
- 不能让她成为"阿湄2.0"——她的黑化有她自己的语气，是克制的、低调的、不张扬的
- 不能让她痛苦地挣扎每一次越线（那会让她软）
- 不能让她越线到"她会后悔"的程度——轻度黑化，她能承受，这是重点

---

### 阿湄弧：情愫→转灰→林夙死→杀疯

**这是全书情感分量最重的一条弧，分三段执行。**

#### 第一段：情愫期（当前进行中）
阿湄有反派血统/基因，一直在压制。她的"命里空缺"连接这件事——被归隐谷痕迹抹去的，可能正是她真实的来历。

她喜欢林夙，是她**第一次选择为了一个人往灰里走**。

伏笔写法：她帮他的某些事，超出了她应该帮的边界。她知道那个边界在哪里，但她越过了，没有解释。读者慢慢看出来：她在为他做一些她本来不会做的事，代价由她自己承担。

这一段，她的反派本性是**压着的**——偶尔一个眼神，一句话，露出一点底色，但立刻盖回去。像一把插在鞘里的刀，读者知道那把刀很利，但她没有拔出来过。

**阿湄线情愫期锚点（已有）：**
- ch119：橘子的温度（她手上的热）
- ch123：碗沿的温度（同上，第二次）
- ch126：「就是想看看你」（直接，不取回）
- ch129：她问了苏挽那句话，带走了，没有说给他听

#### 第二段：林夙"写死"（关键节点，预计beat 3末或S4开）

> **【硬规则·读者反馈回炉】这场"死"必须轰动，绝不许不明不白。** 旧稿假死写得太糊、太轻、太私下——读者第一反应是"他到底怎么没的？"，而不是"我去他死了？！"。重写时必达以下标准，缺一即回炉：
> 1. **当众、有目击、有定论**：不是后院里悄悄没的、不是一句"失踪了"带过。要在一个**所有人都在看的场合**（赛场/宗门大典/长老会对峙/公开处刑式的局）当着众人的面发生，事后全玄苍界都传"林夙死了"，连官方命册都判了死。
> 2. **死法清楚、可信、有重量**：读者要能一句话说出"他是怎么死的"（被谁、用什么、当着谁的面）。机制讲清楚（叶观澜怎么做的局、为什么所有人包括阿湄都信），别留"到底死没死"的含混——含混是悬疑，**不明不白是失败**。
> 3. **有一句/一个可截图的炸点**：一个画面或一句话钉住"林夙死了"这一刻（范文级，像 ch010「倒了」那种重量级单句），让读者那一下心脏停一拍。
> 4. **代价摆在台面**：他的"死"要砸到每条线上——苏挽、阿湄、凌朔、裴无咎、牛阿大各自接到这个消息时的反应，是第三段杀疯/黑化的引信，必须当场点燃。
> 5. **读者与阿湄同步被锤**：读者跟阿湄一起信了他死了（信息差留给叶观澜"留着林夙后用"那一层，不要提前泄底给读者，但"死"这件事的冲击要十足）。
> 写满这五条 = 轰动；少一条就回到"不明不白"。这是全书最大情绪炸点，宁可多铺三章，不许潦草一笔带过。

- 不一定真死——但**对全世界和阿湄而言必须是铁板钉钉的"死"**（假死/被判定已死/当众重伤殒命皆可，关键是"判了死"且人人信），绝不能是含糊的"失踪/下落不明"
- 但必须有一段时间，**所有人都以为他死了，包括阿湄**
- 这段时间里，阿湄的状态：外壳开始裂，压着的东西开始往外走
- 她可能做了一件很极端的事，或者消失了一段时间
- 读者在这段时间里，才第一次看到她**没有在压的样子**

#### 第三段：反派激活·杀疯
- 林夙"死"的消息确认后，阿湄的压制彻底解除
- 她的本性全面苏醒——她擅长的那些事（阴谋、杀伐、消除威胁），全部不再克制
- 她疯，不是因为要复仇，是因为**没有那个人，她就没有不疯的理由了**
- 关键写法：她杀疯的同时，偶尔有一个动作，是他教的/他习惯的/他会做的——那个动作每次出现，读者就心绞一下
- 她的反派状态里，有一层底色，是他的影子

**林夙回来时（若有）：**
- 她已经做了一些收不回来的事
- 她见到他的第一反应，不是哭，不是跑过去——是停住，然后说了一句很普通的话，像什么都没发生
- 那句普通的话，比任何告白，都重

**写作禁区（第三段）：**
- 不能写成"为了他去复仇"（她不是复仇，她是失控）
- 不能让她快速和解（她做的事有重量，重量需要时间）
- 不能让她在他回来时立刻"软回来"（她的杀疯不是表演）

---

### 叶观澜弧 × 阿湄自我觉醒

**这是全书最大的反转，必须分三层建立，才能成立。**

---

#### 第一层：叶观澜 × 阿湄的前史

叶观澜不只是阿湄的信息来源——他是她命数的持有者。

阿湄的"命里空缺"，那张被人彻底抹干净的命格，**是叶观澜抹的**。他以某种方式切断了她与世界账册的连接，让她成为一个不被任何人追索的人，可以做任何事，做完之后，没有记录，没有因果，没有痕迹。

她是他设计的工具——最好用的那种，因为她自己愿意。

这段历史的细节由章节展开，但核心逻辑：
- **叶观澜拥有阿湄的"根"**：她的真实来历、她的压着的反派血统、她能力的来源，都经过他的手
- **盗跖是他的一张网**：阿湄以为自己属于盗跖，但盗跖是叶观澜其中一条线
- **她为他工作过，不是被迫——是因为那时候她没有别的定义**

她跟着林夙做的那些事，是她第一次用自己的判断走，而不是用叶观澜给的判断走。叶观澜容忍了一段时间。然后他决定不容忍了。

**写阿湄前史的规则：**
- 不要一次性说清楚，用碎片，让读者自己拼
- 阿湄自己对这段历史的态度：不提，不是遮掩，是她真的不常想它——那段时间对她来说不是"历史"，是"之前"
- 叶观澜提到阿湄时用的词，要让读者察觉"他对她的了解不止于此"

---

#### 第二层：叶观澜假杀——他的计划 vs. 真实发生

**叶观澜的预判：**
> 阿湄对林夙动了真心 → 林夙是她现在行动的锚 → 拿掉林夙，阿湄失去锚 → 阿湄会在失控中回到他这边，因为他才是她真正的定义

他用假死。不是真杀林夙——他需要留着林夙，以备后用。他需要的是让阿湄**相信林夙死了**，让她压制系统崩溃，然后抓住那个空隙把她拉回来。

**真实发生的事：**
1. 阿湄相信了。林夙"死"的消息传到她那里，那一刻，她所有的压制层同时失效。
2. 叶观澜等的是：一个失控的、哭着跑回来的、可以被他重新定义的阿湄。
3. 他等来的是：**一个已经不需要任何人来定义她的阿湄。**

> **不但没抢回来，还自我觉醒了。**

她的黑化不是哀痛的溃堤，是**她第一次没有在压着自己**——压了那么久的那把刀，终于拔出来了，她发现那把刀比她以为的利得多，而且她不想还回去。

她不回叶观澜那里，因为她现在是她自己的东西，不是任何人的工具，包括她自己以为她可以依赖的林夙。

**叶观澜的计划为什么失算：**
- 他以为阿湄喜欢林夙是因为"软了"、"被驯化了"，拿掉林夙，那个软会消失，剩下一个可用的阿湄
- 他不知道：林夙给的不是驯化，是**一个阿湄第一次愿意承认自己在意一件事的空间**。那个空间没了，消失的是"愿意压着"，不是"阿湄本身"
- 她压着的东西，本来就是她的一部分，不是林夙给她的——林夙只是给了她不想压着的理由。理由消失，"不想压"的感觉，反而更强了。

**叶观澜面对自我觉醒的阿湄：**
- 他不怕她——他了解她，他设计过她
- 但他第一次发现，他了解的那个阿湄，只是她被压着时的阿湄
- 她完全苏醒的样子，超出了他当初的设计范围
- 她不再是他的，但她也不是任何人的——这个局面，是叶观澜制造出来的，他必须负这个代价

**叶观澜×阿湄 写作规则：**
- 叶观澜面对自我觉醒的阿湄，不能写成简单的"被打败"，是"对视"——他是她来的地方，她是他失算的证明
- 两人之间的张力：他比任何人都了解她，但他了解的是她过去的样子；她的现在，他不了解
- 不能让阿湄在叶观澜面前崩溃——她不欠他解释

---

#### 第三层：林夙活着回来

阿湄在"林夙死了"的那段时间里，做了一些事。那些事有重量。

他回来了。

**她见到他的反应，不能是哭，不能是扑过去。**

她停住，看着他，说了一句很普通的话——像什么都没发生过。

那句普通的话，是那段时间所有重量压在一起的出口，它看起来普通，因为她不知道怎么说别的，或者她知道说什么都不够，所以她说了最轻的那句。

读者在那一句里，要看见她做的那些事的影子。

**写作提示：**
- 她做的那些事，林夙不是不知道（他有自己的消息）
- 他看见她说那句普通的话时，他知道那背后是什么，但他也没有立刻说破
- 两个人都知道，但都先说了别的——这个空间，是两个人给彼此的

#### 前置伏笔要求（当前章节需要埋）

**以下内容必须在假杀事件发生前，用至少3章的篇幅，分散建立：**

1. **叶观澜对阿湄的"了解感"**：他提到阿湄时，用的词，要让读者觉得他对她的了解不止于"信息渠道"——是某种更底层的熟悉

2. **阿湄的某次"越线"，被叶观澜看见**：他旁观她为林夙做了某件超出正常范围的事，他的反应——不是警告，是某种"哦，到这一步了"的评估。读者要看见他在重新算什么

3. **阿湄偶有一个动作/眼神，是压制的底色在往外走**：她自己没有意识到，但林夙（和读者）看见了那一秒。那一秒，是她苏醒时的预告

4. **叶观澜主动找过林夙一次**：不是杀意，是考量。他在看林夙是什么，能不能留着。这场见面的气氛，要让读者在事后（假杀发生后）回想起来，意识到那就是他决定动手的时机

5. **阿湄说过一句让林夙没接话的话**：类似"我做这种事，做惯了的"或"你不用管我之前是什么"——林夙当时没有接，读者到假杀事件后，才明白那句话的重量
