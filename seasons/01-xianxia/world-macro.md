# 玄苍界 · 大环境

> **一句话**：这是一个把"命"做成账本、再靠篡改账本来统治的世界。底层灵气稀薄（经济稀缺），上层靠"命不可改"的信仰维稳（意识形态），而当权者私下天天改命（文过饰非）。镇狱之渊——被镇住的那座狱、被埋掉的那段渊——就是这套体制脚下不许翻的旧账。

---

## 〇、总枢纽：命册体（连接六大块的轴）

- **命册**：天地众生的"前世今生死生节点"被记录在册。**知命者**（全界八位，已立设，见 ch083）是活的册子，天生能望见他人命格。
- **命册 = 户籍 + 财产证 + 判决书 三合一**。一个人的根骨、阶层、能调动多少气运，出生时一"批命"就定死——这既是普查，也是种姓分配。
- **可写即可改**：能记录的东西就能被涂改、被买卖、被抹除。于是命册不是天理，是**权力的接口**：
  - **知命者**——能读、能卖记录（姜玉衡把林夙上一世的死卖给方玄，ch083）。
  - **归隐谷**——能抹除记录，让一个人从世界的账上消失（阿湄的"命里空缺"就是被抹的疤）。
  - **复盘之眼**（林夙）——能读到**未被涂改的原始版本**。所以他不是开了挂，是个会走路的"原始档案"，对整台文过饰非的机器是活体威胁。
- **镇狱之渊**：书名即体制的承重墙。被"镇"住的旧史里，藏着这套命册体当年是怎么立起来的——一段所有当权者都靠它合法、却谁也不许翻的渊。

> 写作用法：六大块不是设定百科，是**六种压在角色身上的力**。每一块下面都给"它怎么逼角色"，正文只写逼出来的结果。

## 一、经济 · 稀缺与命债

**经济基座**：玄苍界边陲灵气稀薄（承 world.md）。稀缺是一切的发动机——灵气不够，所有人都在抢同一口气。

- **三层货币**：灵石（明面通货）→ 灵脉（生产资料，被宗门垄断的"田"）→ **气运/命数（硬通货）**。真正的富人买的不是灵石，是命——延寿、改运、给子孙挪一个好命格。
- **宗门即地主/财阀**：青锋宗一类把持灵脉，向附庸小族收"孝敬"=地租（钱执事收孝敬看人下菜，已立设）。林家、苏家是食物链最底的寒族，连被收租的资格都要争。
- **命债（贯穿主线的核心经济词）**：欠的不只是钱，是命。卖身、卖运、卖子女抵债是常态。**林夙"这一世只算账"——他的复仇被框成讨债，本就该是这套命债经济的镜像**：他讨的是一笔被全家、被宗门赖掉的命债。
- **蛊场（叶观澜暗产）= 把人炼成资源的工厂**：童年阿湄"入蛊场前被人采过"（ch061）——人在这里是原料。这是大环境最冷的一格：底层的命被采、被炼、被结算。
- **盗跖（叶观澜暗网）= 影子市场**：买卖赃命格、违禁蛊、被抹过的身份。明面命册管不到的交易，都在这里走。

> **怎么逼角色**：钱执事势利、苏震山拿女儿当筹码、王姨娘捧高踩低——不是他们坏，是稀缺把人逼成秤。写他们时，让读者看见那只看不见的手在背后掐着他们的脖子。

## 二、政治 · 没有皇帝，只有册

**统治形态**：玄苍界不靠一个皇帝，靠**命册官僚体**统治。权力金字塔：

```
知命者（八位·记录权的最高持有者）
        │
归隐谷（抹除局·体制的"深水区"，专管让人消失）
        │
各大宗门（青锋宗等·区域诸侯，向上称臣、向下收租）
        │
寒族世家（林家/苏家·被秤量的一层，无记录权）
        │
黑户/被抹者（阿湄这类·账上不存在的人）
```

- **归隐谷 = 深层国家**：它不治理，只**删除**。一个人碍了事，不杀——抹掉他的命格，让他活着却不存在，没有记录、没有因果、没有人能追索（CLAUDE.md §九叶观澜弧已立此设）。这是最高政治暴力。
- **叶观澜的真实位置**：表面是不良于行的可怜人（全城惋惜他的腿，CAST），实为蛊场与盗跖之主——他是**这台抹除/采炼机器在边陲的操盘手**。Daddy×Girls 框架里他是引力中心，根就在这：他掌握"让人存在或消失"的权力，所以几条女性线都被这份权力拽着。
- **赤渊**：满口宗门大义，恶事只为一个死人（CAST）——他是被体制意识形态彻底内化、又被私情反噬的"忠诚执行者"标本。

> **怎么逼角色**：阿湄一切行动绕回叶观澜，因为他握着她的"根"（她的命格记录）=政治人质。苏挽后期黑化，本质是**对这套官僚体失去信任**：规则（册）没保住她在乎的人，她于是开始在册外行动。

## 三、文化 · 命定论与回声

**主流信仰：命定论**——"批命定终身，根骨定贵贱，命不可改"。

- **批命/验根骨**：婴儿出生即批命验骨，当众宣读（参考 ch061 管事验阿湄根骨那场）。这是文化仪式，也是种姓盖章。**林夙"废物庶子"四个字，是被这场文化判决盖死的**——不是他不行，是册说他不行。
- **茶肆说书 = 民间媒体 / 世界的回声**（说书人已立设）：一面替当权者把"天才—废物"的秩序唱成天经地义（propaganda），一面又是流言的温床，能把主角的传闻越传越神（subversive）。同一张嘴，两副作用。
- **节庆与宗祠**：拜的是"安于命"。逢年过节强化"各安其位"，宗祠是把命定论代代焊死的地方。

> **怎么逼角色**：林夙归来扮猪吃虎，不只是战术——是**他必须戴着文化判他的那张"废物"脸活动**，每一次隐忍都是在文化判决下苟住。读者爽的是他终将撕掉那张脸；高级读者读到的是"命定论被一个看得见原始档案的人证伪"。

## 四、意识形态 · "命不可改"的双标

这是大环境最锋利的一格，也是全书主题的引擎。

- **统治话术**：对下，"命不可改"——你穷你贱你死，都是命，认了。它让所有人安于其位，是最省力的维稳。
- **隐藏现实**：对上，**当权者天天改命**——知命者卖命、归隐谷抹命、富人买运挪命格。
- **核心矛盾（一句戳破）**：**愚民信"命定"，当权者改"命册"。** 谁能改命谁就是权力，谁信命不可改谁就是被治理的对象。
- **异端 = 任何能改命的人**：林夙（读原始命册）、阿湄（被抹后反而脱出册外）——他们都是这套意识形态眼里的危险品。林夙的复盘之眼若曝光，等于公开宣布"命可被读、被改、被伪造"——这会拆掉整个金字塔的合法性。

> **怎么逼角色**：不写论文，写信仰崩塌的瞬间。苏挽的黑化曲线就是一个人**从信"册能保护我在乎的人"到亲手在册外行动**的全过程（CLAUDE.md §九）。每越一次线，是她意识形态退一格。

## 五、文过饰非 · 被涂改的历史

**命册可被涂改 → 历史本身就是一本被反复删改的账。**

- **官方的过去 ≠ 真实的过去**。知命者售命（ch083）、归隐谷抹命，都是在改写"已经发生的事"的记录。久而久之，世人记得的历史是删改版。
- **林夙的复盘之眼 = 活体未删改档案**：他能看见每个人"前世结局/怎么死的"（world.md）——即原始版命册。在一个靠涂改维稳的世界里，这种眼睛是最危险的政治存在：他走到哪，哪里的官方叙事就可能对不上账。
- **阿湄 = 文过饰非的人形代价**：她的"命里空缺"不是天生缺失，是被叶观澜（经手抹除，CLAUDE.md §九）**删掉的一页**。一个被从历史里删除的人——既是体制最彻底的受害者，又因此成了不被任何因果追索的最好用的工具。她的整条弧，就是"被删除者夺回自己被删的那一页"。
- **镇狱之渊本身**：被镇、被埋的那段旧史（识忆之线，arc.json"起"beat 已起），是全书最大的一笔被涂改的账。决战揭真相＝翻出这本被镇住的旧账。

> **怎么逼角色**：文过饰非不靠旁白交代，靠**对不上账的细节**——一个老仆记得的事和官方说法不一样、一块墓碑上的名字被磨过、说书人某段词被禁。让读者自己发现"历史被人动过手"。

## 六、历史形态 · 旧世 / 镇压 / 当下

**分期（供伏笔与决战回收）：**

| 期 | 名 | 核心 |
|---|---|---|
| 旧世 | 识忆之线浮起前的世界 | 命册体如何立起来的源头；镇狱之渊事件发生于此期末 |
| 镇压期 | 渊被"镇"住 | 当权者用归隐谷+知命者把那段历史抹平、镇封，建立"命不可改"的新秩序 |
| 当下 | 林夙归来这一季 | 体制成熟、僵化；稀缺加剧；一个带原始档案之眼的人回来了 |

- **"镇狱之渊"既是地名也是国策**：把某座狱/某段渊镇住，是这套政体的立国行为。所有当权者的合法性都建立在"那段历史不许翻"之上。
- **林夙的复仇 = 私仇接通公史**：他从讨自家命债起手，越查越深，终将撞上"为什么我家会被这样对待"的体制答案——私人复仇线（男频升级）与翻旧账（揭文过饰非）在他身上合流，正对 CLAUDE.md §零-6 "男频升级线与女频情感线在他身上交汇"的根。

## §写法戒律（大环境如何落地，不许违背）

1. **大环境只能被"感到"，不能被"讲到"**。禁止旁白科普经济/政治/制度。对照 chronicle 历次审稿硬伤（ch071/ch173/ch263 全栽在"把判断直接说破"）——大环境一旦被解说，立刻变扁。
2. **一笔写两层（雅俗共赏，CLAUDE.md §零-5）**：表层是角色的具体处境（爽/虐），深层才是制度的影子。秤砣范本——林崇那杆"称量儿子的秤"，一个家庭细节，扛起整套命定论。
3. **通过损失体现压力**：制度的重量＝它让角色失去了什么、不得不变成什么。写王姨娘的尖刻、苏震山的算计、钱执事的势利时，让读者看见稀缺与册在背后掐着他们。
4. **文过饰非靠"对不上账"**：永远用细节矛盾暗示历史被改，不用"原来当年……"的全盘揭晓（决战除外）。
5. **每条线认领一块**（与 §零 多线并行配套）：
   - 林夙线（男频升级）→ 经济·命债 + 文过饰非（翻旧账）
   - 叶观澜线（反派布局）→ 政治·抹除 + 蛊场经济
   - 阿湄弧 → 文过饰非的人形代价（被删的一页）
   - 苏挽黑化 → 意识形态崩塌（册外行动）
   - 宗门权谋线 → 经济·地租 + 政治·诸侯
