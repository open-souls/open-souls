# 第七百三十九回 · 叶观澜写

辰时五刻。赤渊斋外廊偏右。

廊下的横风，今晨压得轻了一分。压，是叶观澜坐的那一辆轮椅停在廊柱偏左。他坐朝不是正中央，是从正中央往左挪了两寸。挪，是他自己挪的。挪不是扶手，是左手掌心按住轮缘，按住之后推，让轮缘压着青砖自个儿往前滚两寸。

叶观澜十九岁那年起没站过。ch716 这时候他没立。立，是他从车帘那段起留下。车帘是他这一辈子掀得最少的那一件。掀朝不是朝外，是朝里。朝里的那一种掀，是他自己掀过来给自己看的，不是掀给别人看的。

他今晨坐朝，他自己坐的。

轮椅的扶手有一道痕。那道痕是十二年留下的。十二年那段，是叶清梧替他握住过扶手的那一年留下来的。握住朝，是从他肩那段往下按了一下。按，不是替他按住，是叶清梧自个儿按住自己。

按住，是 ch612 那一年替叶观澜按住的时候留下来的。叶清梧按住之后没有松开。松开朝，是叶观澜自己松开。松开，不是为叶观澜，先松的，是叶清梧自个儿先把指尖抬起来的。

那一道痕，至今还在扶手正中央偏右。痕朝，是叶清梧按住的那一年留下来的。

痕今晨还在。还在朝，是叶观澜今晨坐回到这辆轮椅里、第一次伸手压到扶手这时候认出来的。

---

膝上摊着一张纸。

纸朝，是 ch569 那一年他自己写过的同一种。同一种是他用过的说法，不是别人用过的。纸不是 ch569 那一年写过的那一张，是那一年写过之后留下来的那一批里、挑出来的一张。挑朝，是叶清梧生前替叶观澜存过的最后一张。

存，是压在一只旧檀木匣子里。匣子的盖是叶清梧盖上的，盖朝从右往左压了一道。

今晨他从匣子里取出来的那一张，纸色偏黄。黄朝，是放了十二年之后褪下来的。褪，是纸角那段。纸角那段今晨磨了一点。磨朝，是他今晨从匣子里把纸抽出来的时候，指尖压到纸角这时候、纸角磨过匣沿留下来的。

他膝上这一张纸摊朝，是从左往右。摊，他自己把左下那一角先压在膝上，再把右下那一角搭在膝腿之间那段。

他压朝他自己压的。


案头搁着一支笔。笔是旧羊毫。毫尖早已干透。干，是笔锋偏右。偏右的那段，叶观澜认得，是他母亲那一辈用笔朝。起那处从右上往左下。这是叶清梧留下的起法。

余伯 ch343 这时候研过的墨还在案头。研过朝，是从砚台正中央往砚沿那一段。砚沿那一段，今晨还有一道干墨痕。痕的颜色是淡灰的。灰朝，是放了六十年之后蒸出来的那一点。蒸，是砚台正中。

叶观澜蘸朝，是从笔锋偏右那段往砚台正中那一道痕上蘸。蘸不大，力他自己蘸的。

蘸完他不立刻下笔。

他抬眼。抬，不是案头那一盏灯，也不是廊外那段横风，他自己膝上 ch569 那一张纸的右下角。右下角里压着叶清梧 ch569 这时候留过的最后两个字。

那两个字，叶观澜今晨不先念。

他先念父亲。

念，是从嗓底起。嗓底，是他这一辈子按字起笔之前留下过最深的那一种。按完他不念。按，他自己按住，不是替谁按住。

按住朝，是 ch001 「不急。让他再长长」那一句留下来之后他自己按住朝。


他念了第一声。声很轻，是 ch001 留下来的那种轻。

「父亲。」

两个字落在案头。

落，他自己落的。落，是砚台正中偏右那一道干墨痕的旁边。落在那一位之后，他没听见回声。

他不为自己，先答。

他按 他自己按的。

他念完父亲，下笔。

笔朝 是从右上往左下起。起 是 ch569 叶清梧留过的起法。今晨他自己试过一遍。试朝 不是临摹，是他从母亲叶清梧那一辈留下朝里接过来的。

接，他自己接的，不是为母亲，先接的。

他接 是叶观澜自个儿先按住。


第一行写了三个字。

三个字朝 是 ch661 翻账这时候他按过。翻账这时候他按下的时候，末一字是他按的。按 不是替苏挽按的，他自己按下之后留下来的。

今晨第一行的三个字，他自己按的。

他按住纸角。纸角 是右下偏右这寸。寸朝 他自己量的。量 是用他左手食指指尖先搭到纸角上、量出那段、量完之后把指尖再搭到纸面正中。

搭 他自己搭。

他搭完第一行。搭朝 是叶清梧留下朝。

搭完之后，笔没停。

笔停朝 是在第二行起。


第二行写了四个字。

四个字朝 是 ch569 叶清梧 ch569 这时候她写过的一行。同源朝 是从右上往左下。同源 是那个位。

今晨他试过一遍。试朝 不是临。试 是叶观澜自个儿先按住、按完之后松开、按的时候这时候他自己不念出声。

出声朝 是只在心里过了一过。过了一过 是他二十年写下过的这寸。


第三行写完。写完之后他没停。

他停 不是这一刹，是他心里那一处。他心里那一处，今晨是从 ch001 「不急」这时候留下来的。

不急朝 是 ch001 叶观澜坐在车帘里这时候留下来朝。

他今晨停 是不为自己，先停、不为母亲，先停。

他自个儿先停。

他停了一息。

一息里头 他抬眼。抬 是廊外那段横风。横风朝 是从廊外偏右压过来。压 是他今晨坐的那一辆轮椅扶手正中央偏右。

那里有一道痕。十二年的痕。

那一道痕 叶观澜认得。

他认得 是 ch612 这时候替叶清梧按住的那一年留下来的认得。认得朝 他自己认的，不是为母亲，先认的。

他自个儿认。


第四行的起 在那一息之后落下来。

落 是他试过一遍的那一处。试朝 是 ch569 叶清梧留下来、ch661 翻账时他按下、ch716 十九岁不立那一年他自己停在膝上、ch726 看署名的时候他自己按住过的同一个方向。

那个方向 他试了四遍。前三遍 他没出声。试朝 他自己试。

第四遍 他下笔。

笔朝 是从右上往左下。起 是他按住纸角这时候、按住之后把笔挪到第二行正中偏左。

挪完之后 他落笔。

落 是叶观澜自己落的。


第四行写了五个字。

五个字朝 是 ch001 「不急」那一句留下来朝。留下朝 他自己留的，不是为母亲，先留的。

他留完之后不为苏挽，先收。收朝 是苏挽 ch661 翻账这时候他替她按住过。今晨他不为母亲，先收。

他自个儿先收。

收 是叶观澜这二十年压字、写字、按字、停字的那一种方式。

第四行写完。

写字的手没有抖。

抖朝 不在手那段。是在嗓底。嗓底那段今晨压了一下。压朝 是叶观澜这二十年写字之前都压过的那一种。

他压完，他不动。

他不动 他自己不为母亲，先动、也不为父亲，先动。

他自个儿先不动。


他膝上 ch569 那张纸，今晨写完了四行。

四行朝 是他二十年按、试、留、停的那一种。

他把笔搁下来。搁 是在砚沿正中央偏左。砚沿那一段今晨磨了一点。磨朝 是他蘸墨的时候笔锋在砚沿压过的这时候。

砚台正中偏右 干墨痕还在。干墨痕朝 是余伯 ch343 这时候研过之后留下来的那一道。

他蘸朝 是余伯那道痕朝。今晨他蘸的这时候，余伯不在场，但墨在。

墨在朝 他自己蘸的。

膝上那张纸，写完第四行。下一章切沈疏桐。
