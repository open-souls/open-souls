# 第七百三十三回 · 林夙回

辰时三刻。临安第三街旧档室。

晨光从木窗缝斜压进来。光是淡的，带着青砖的寒气，一寸一寸地往档室正中那张旧木桌上压。桌沿那道痕还在——是六十年那一刹叶清梧研过墨留下来的，深半分，磨得发亮。墨香早就散了。桌面上只摊开一封信，是 ch713 阿湄从宿州写来的那一封。信封正面右下角「宿州」二字墨已干透，干在卯时初。

林夙今晨第一次回。

他不是昨夜走的。昨夜他坐在档室正中，坐到灯火芯灰落了一刹，又坐到更漏把寅时点完。今晨他从门外进来时袖口卷了一折——是夜露卷的那一种，不是他自己卷的。鞋底压过门沿那道木槛，门槛偏右一寸，门槛上那道旧痕是 ch044 那一年他自己进门时踩出来的。

他坐下。坐得慢。坐在档室正中偏右那一寸——跟昨夜同一个位。

桌面那一封信，他昨夜按住过一遍，今晨他没有立刻抬手去按。他先把右手五指搭在自己膝上，按了一按，按置是膝面那一道旧痕。痕是他 ch502 那一年自己按住过的——六十年、六十年过去了，那道痕还在。

他自己也没料到，今晨他要做的事，不是按信。是开口。

林夙喉咙动了一下。

他喉底开，开口极轻。极轻是 ch726 他自己按住「梧」字末笔那刹他声音落在袖底的那一种轻。

「叶清梧——是谁。」

他问的不是余伯。

他问的是他自己。

他听见自己把这句话说出来在信纸末行右下角那一截「梧」字末笔那一寸。他说出来的方式是他自己按住过的。不是问余伯。是问自己——那三个字背后那六十年，是从谁那里来的。

他没有替余伯先答。也没有替叶观澜先认。也没有替沈疏桐先答。也没有替阿湄先回。

他替他自己问。


余伯坐在东边旧木桌，离他三尺。

余伯的手里没有按东西——余伯 ch727 接住的那封旧函，今晨压在桌沿偏右那一截，余伯按置是掌心底那半寸。余伯听见林夙问，余伯没有立刻开口。

余伯喉底动了一下。

余伯没有替林夙先答。

林夙看见了。余伯不动就余伯 ch692 第一次正面说叶清梧那一种动。那一种动，余伯压了六十年，今晨压得最稳。

林夙也没有立刻再开口。

他今晨第一次主动开口、第一次把那个名字念出来、第一次反推那一刹他替自己先问的那一种——他问完，他不动。

不动是 ch713 他自己看信 + ch726 他自己看署名 + ch723 他自己看字认出同一刹，那一种按完又按、按完再没松过的那一种不动。


桌面那一封信还在。

信纸末行右下角，「叶清梧」三个字还在。字老了六十年。字形细得均匀——均匀的是 ch085 那一刹叶清梧半个身位他自己研过的那一种均匀。墨干了。墨香也干了。干的位置是「叶」字起笔那一寸。

林夙看着那三个字，看了有一会儿。

他没有再伸手去按。

他看着的方式，是他 ch634 那一年他自己念过名字那一刹留下来的那一种看——是他认出来的同一种。

他认出叶清梧三个字了。

认出来在「梧」字末笔那一寸。

认出来的来路——是他 ch636 那一年认林家那一刹留下来的同一种认。林家他认得。叶清梧——他认不出来。但那三个字底下压着的事，他认得。

他今晨第一次反推。

反推朝叶清梧三个字背后那六十年，是从谁那里来的。是余伯替叶观澜按住过的那六十年，还是余伯替叶清梧先研过墨的那六十年，还是余伯 ch612 替叶观澜按住过的那一刹他自己留下来的那一种。

他不动。他问完他不动。

晨光又压了一寸。

桌面那一封信——他今晨第一次看见。他昨夜按住过一遍。今晨他看见的方式，是他自己先替他自己按稳再问的那一种。

他按就他 ch502 那一年他自己握住过的那一种按。

他今晨第一次回档室、第一次主动开口、第一次问。


档室东边偏右那一截，余伯坐着。

余伯从昨夜坐到今晨。坐得稳。余伯 ch727 接住那封旧函之后，他没有起身去添过茶，也没有起过去添过水——炉里那点柴没有点，余伯也没有点。茶凉了。茶是昨天剩的，凉在案头一整夜。

余伯听见林夙问，余伯没动。

余伯不动就余伯从他 ch569 收叶观澜那封信那一刹留下来的同一种不动。六十年压着不动，今晨压得最稳。

「你认出来了？」

余伯没有替林夙先答。余伯替林夙接住这个问题，但答案，余伯让林夙自己说。

他不翻的来路——是 ch700 他自己回到 ch085 不翻 + ch711 他自己看见署名不翻 + ch718 他自己看旧墨不替林夙先看，那一种按完又按、按完再没松过的那一种不翻。

档室偏右那一截，更漏声还在。

漏声极轻。轻到档室里头的人是听不见的——但林夙听见了。他听见的方式，是他自己耳底下压着的那一种轻。压的来路——是 ch634 他自己念名字那一年他自己留下来那一种轻。


压的位置是档室正中偏右一寸。压朝「梧」字末笔那一寸。
