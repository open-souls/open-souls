# 上线评分表（满分 14，< 9 退回重写一次）

每项 0 / 1 / 2：
- 钩子：结尾有没有让人想追下一回的悬？
- 爽痛：这回有没有明确的情绪兑现（爽 / 痛 / 转）？
- 反差：有没有用到某角色的「说 vs 做 / 表 vs 里 / 被逼到墙角」？
- 拉扯：关系靠推拉前进，而不是把张力解决掉？
- 记忆点：有没有一句能被截图 / 转发的话，或一个强画面？
- 代入：有没有一个清晰的视角让读者钻进去？
- 新：桥段是新用 / 混搭，还是套路平铺？
