# 流量密码 · 桥段库（学形状，不抄原文。审校把高分桥段提拔进这里）

## 反差（嗑点核心，从角色 fracture / under_pressure 生成）
- 表里不一：高冷/心慌、笑面虎、人前神人后怂。读者爱的是那道缝。
- 双标：对全世界一个样，只对 ta 破例。
- 破防/黑化：温柔的人被逼到墙角露出狠，狠人露出软。

## 中国网文
- 打脸/逆袭：被看轻 → 反手碾压。每回一个小爽点。
- 扮猪吃虎 / 马甲：藏身份，露的那一刻最爽。
- 先婚后爱 / 破镜重圆：错位的亲密，慢慢回温。
- 世事不公 → 反抗：先把不公写狠，反击才爽。
- 团宠 / 苟道：被全员偏爱 / 猥琐发育。

## 日本动漫
- 異世界転生：换世界重新开局（你的赛季机制）。
- ツンデレ / ヤンデレ：傲娇 / 病娇——极致反差。
- デレ瞬间：冷的那个突然破功，一回攒一次。
- 仲間 / 羁绊：临时凑的人，变成不能失去的人。
- 伏線回収：早埋的细节后面爆，"原来如此"。

## 西方
- enemies-to-lovers：欢喜冤家，张力拉满。
- morally-gray antihero：灰色主角，没有纯好人。
- slow burn：慢热，憋着不挑明，越憋越上头。
- redemption / fall：救赎或堕落弧。
- cliffhanger serial：每回断在最吊处（狄更斯连载术）。

## 通用机关
- 钩子：每回结在一个没解开的悬。
- 爽/痛：每回给一个明确的情绪兑现。
- 记忆点：一句能被截图转发的话。
- 黄金三章：开季前三回钩子密度拉满，别铺设定。
- 拉扯 > 露骨：张力比直给更上瘾。
