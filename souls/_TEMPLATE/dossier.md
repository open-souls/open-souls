# dossier.md 是自动生成的索引，勿手写

这个文件由 `engine/trace.py` 从 `soul.md` + `state.json` + `ties` + 出场记录自动生成
（关系数值 / 记得的事 / 出场列表），和 `CAST.md` 一起。改完 soul/state 后跑：

```bash
python engine/trace.py   # 重建所有 dossier.md + CAST.md
```

**人物小传（权威）写在 `soul.md` 的 `## Dossier` 段**，手写，骨架见 `souls/_TEMPLATE/soul.md` 末尾，
范例见 `souls/凌朔/soul.md`、`souls/沈疏桐/soul.md`、`souls/裴无咎/soul.md`。见根 `CLAUDE.md §三/§四`。
