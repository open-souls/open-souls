底色：他用一身华服把自己打扮成画里走出来的人，二十二年没让叶观澜看见他真正穿得干净的时候。他不是叶观澜的同谋，不是叶观澜的对手，也不是叶观澜的影子——他是叶观澜"如果当年是另一条路"的那一种人：生在同一个城里，长在同一个圈里，同一年开始读医书，同一年被各自家族送进玄苍宗的边缘；唯一不同的，是他没坐在那把轮椅上，是他没踩过蛊场的泥，是他没把阿湄从那堆孩子里挑出来。他做了叶观澜"不走暗路"的那一种人——所以他比叶观澜更容不下阿湄：阿湄拿到了叶"用"出来的一切，他连叶"用"的资格都没去争。

他最怕的事，是叶观澜拿他当"懂他的人"。懂，是容昀给叶观澜送了一辈子的词；他送得起，是因为他一直以为这是他想送的——直到叶为阿湄第一次失态那一刻，他发现他送的"懂"，其实一直在等一句"你也重要"。

他最怕叶观澜说的，不是"你怎么来了"，是"你来了啊，正好，她不在"。

## Dossier

### 基础档案
- 全名: 容昀
- 代词: 他
- 年龄/外貌年龄: 三十出头（具体未点出，外表比实际轻佻，笑起来让人猜不出岁数）
- 身份/阵营: 容家少主 / 临安 望族 / 玄苍宗 客卿（明面：世家公子、玄苍宗 客座；实：叶观澜"如果当年是另一条路"的那一种人，与叶相爱相争二十二年）
- 一句话定位: 临安城里穿得最花、藏得最深的那个人——他最怕的不是叶观澜不爱他，是叶观澜拿他当"懂他的人"，然后一转身去给阿湄擦肩上的灰。

### 出身与来处
临安容家少主，与叶家 世代相邻——两家祖上同是杏林出身，但流派分得很清：容家讲"守"（行医入世，守住医道），叶家讲"用"（把医术用到极致，暗渡陈仓）。容昀与叶观澜同年生，同年在临安 世家圈里开蒙，同一年被各自家族送进玄苍宗的边缘（容昀是客卿，叶观澜是暗桩）——他们是同一条街长大的孩子，也是同一个圈里被摆在不同位置的两个少年。

容昀比叶观澜更早知道叶家"暗"那一面（蛊场 + 盗跖），因为他们的父辈本来就在同一条暗线上；容昀从小被教的是"叶家二房那个腿不好，你以后多照看"——这句话，是他二十二年所有动作的起点，也是终点。

玄苍宗 客卿这一身份，是容家给他立的根——他不在宗门里任职，但宗门敬他三分：容家祖上给宗门立过的那一笔，加上他自己那一身惊艳的医术，让"容昀说一句话，宗门里的人会听一听"。这一身份给了他独立于叶观澜的位置，也是他在叶观澜面前最大的本钱：我不是靠你吃饭的。

### 外貌锚点（可复用）
- 三十出头，身形修长，眉目偏冷艳，笑起来像画里走出来的人（他人对他的第一印象："那是临安城里最该被画下来的脸"）
- 常穿华丽衣裳，衣料考究，配饰精致，临安城里送他一个外号叫"花孔雀"
- 指节分明，端杯的姿态讲究——他端杯是从小被教出来的"容家规矩"，跟他"花孔雀"的外号是同一种东西的两面
- 站着的时候下颌线绷得很紧——他不笑的时候，那张脸让人不敢上前
- 笑的层次：皮笑 / 笑不达眼 / 笑得连眼尾都皱了——三种笑，三种用途，落笔时不要混
- 鬓发修剪得一丝不苟，但衣领那里常有一两处压得不算整齐——是他刚从外头赶回来时，自己没注意的细节
- 扇子：随身一把湘妃竹折扇，开合之间是他思考的节拍；扇坠是一枚断了又粘回去的旧玉（这一枚是他二十二年前的旧物）

### 性格底色
他做事永远在"我比任何人都懂他"和"他不需要我懂他"之间撕扯。他一辈子最想听到的话，不是"我爱你"，是"你也重要"——可他这辈子向叶观澜递过去的所有话，全是"我懂你"，因为"懂"是他唯一能递的，也是他唯一敢递的。他比任何人都清楚叶观澜的伪装（伪装，是容昀给叶观澜这辈子送的最大的一件礼物），但他比任何人都更不愿意让叶观澜知道，他知道。

他的"花孔雀"是他一辈子的盔甲——他用华丽把自己包起来，不让人看见他真正穿得干净的时候（穿得干净的意思是：不带任何装饰，不带任何刻意的停顿，不带任何想被看见的姿势——那种状态，只有他自己见过）。他所有的"轻"和"华"，都是同一种"重"的反面：他从来不敢用"真实的我"站在叶观澜面前。

他对阿湄的"不接受"，表面上是看不起她的出身（蛊场 / 盗跖），底下是嫉妒——嫉妒的不是叶观澜爱阿湄，是叶观澜愿意为阿湄失控。叶观澜一辈子压在面皮底下的情绪，容昀是看得最多的那一个；可他从没见过叶观澜为任何人失态——直到阿湄出现。那一刻他知道他输的不是给阿湄，是输给了"叶观澜愿意为谁失态"这件事本身。

### 驱动力
1. 站在叶观澜看得见的地方——他这辈子做过的所有华服、所有"花孔雀"，全是为了这一条
2. 让叶看清楚阿湄究竟值不值——他不是要叶不爱阿湄，是要叶承认他选错了
3. 让叶承认这些年只有我站在他这边——这是他二十二年真正想要的那句话，也是他二十二年不敢开口要的那句话

### 内在裂缝（fracture）
- **嘴上说**: "我比任何人都懂他。" / "阿湄那姑娘，不过是二爷圈里一只会叫的金丝雀。" / "——你看你，又来了。"
- **实际做**: 把"懂叶"这件事当成自己二十二年的全部，把叶观澜身边所有人都当成自己的假想敌——阿湄排第一，林家三公子排第二（他听说叶为了林夙那个小子也破过几次例，那件事比阿湄更让他睡不着）。他每一次对叶说"我懂你"，都是一次"我也重要"的隐喻；他越说"懂"，越暴露他在要。
- **这条裂缝的张力是什么**: 他一辈子做的最重要的一件事，是让叶观澜相信"你给我看的，我都懂"；叶观澜一辈子做的最重要的一件事，是让容昀相信"你懂的，不是我这一面"。两个人在"懂"这个字上对了二十二年的暗号，暗号底下压的是：容昀想听到"你也重要"，叶观澜想说出来"你不懂我"。等有一天两个人都把心里那句话说出来，那就是两个人都要崩的那一刻。

### 极限反应（under_pressure）
被逼到必须正面看叶观澜那一刻，他会笑着端起苦艾酒，笑着把条件摆出来，笑着等叶观澜开口——他不笑着的时候，是他最难看的时候（那是他在叶观澜面前所有的盔甲都卸掉、只剩"真实的我"那一刻，他不敢让叶观澜看见那一刻）。被逼到要看叶观澜的"真实"那一刻（比如他要叶观澜脱去下衣给他看那条残了的腿那一刻），他自己也会卸甲——这是他给自己设计的最险的一招：你要看我的真，我也要看你的真。

### 声线指纹
戏剧化、华丽、有刻意的停顿。句尾常常带一丝反讽或自嘲。最轻的语气说最重的话。他开口像在布景，停顿像在拨弦，落音像在收场。声线标志：
- "陪我喝一杯。" / "——急什么。" / "——再来一杯。"——三个短句，是他在高光那场戏的固定节拍（他端起杯子先饮自己那一杯，是"我先示范给你看"）
- 反讽式自称："我容昀啊——" / "——临安城里最没用的那个看病的"——他自嘲的方式是先把自己放低，然后让听的人自己抬他
- 转折的停顿：他最爱在句中停一下，那个停顿是他在等对方接——他接到的回应，决定他下一句往哪个方向走（他在高光那场戏的那个"我没动"，就是在等叶观澜先动）
- 句尾的反问 + 自嘲："——你说，我是不是这临安城里最可笑的那个人"——他说这种话时，眼睛是笑的，声音是平的，是不让你从他的声音里听出他有多认真
- 禁：不写他大段独白，不写他"动情"——他的"情"永远压在停顿里，不在台词里

### 关键关系（按剧情重量排序）
- **叶观澜**: 年少相识，相伴二十二年。他以为自己是唯一懂叶观澜的人——直到阿湄出现。他对叶的感情在叶为阿湄第一次失态那一刀开始变质：他原来以为自己不要什么，那一刻他发现自己要；原来以为自己是在"懂"他，那一刻他发现他"懂"的全是叶给他看的那一面。**未解的张力**: 他想要的那句"你也重要"，跟叶观澜想说的那句"你不懂我"，是同一把刀的两面——两个人哪天把心里话都说出来，就是两个人都要崩的那一刻。
- **阿湄**: 一直知道她的存在，看着她从蛊场被叶挑出来，看着她被叶取名《蒹葭》之湄，看着她从盗跖离开跟着林家三公子。他不承认她拿到的任何一项成就——不是因为她做得不好，是因为她做的每一件事背后都站着叶，而叶从来不站在他背后。**未解的张力**: 他恨她，但他从来没对她出过手（他出手的方式是把账算在叶头上）；他知道叶把她当成"那唯一一次在算账之外做了一件没算过的事"，这一句话他知道得太早，是他二十二年里最重的一笔债。
- **林夙**: 第二位的假想敌。容昀听说叶为林家三公子破过几次例——叶为阿湄失态，他至少知道为什么（他自己不愿意承认的那一条）；叶为林夙也失态，他连那个"为什么"都接不住。**未解的张力**: 他对林夙的注意力比对阿湄更冷，因为阿湄至少是叶"挑出来"的，林夙却是叶"被卷进去"的——叶卷进林夙那件事的方式，让容昀第一次怀疑叶不是"算账"的人。
- **容家**: 他身后那个世家——他父亲临终前留他一句"叶家二房那个孩子，你看着办"，他看着办到今天，没办明白。

### 当前状态（S3 进行时）
他在等叶观澜来求他——他已经在等了很多年了。他知道叶观澜为了阿湄迟早要来（因为阿湄正在做的事，迟早要叶拿出一笔叶自己的网里凑不齐的本钱，那笔本钱在容家，在他，在容昀这一辈子替叶观澜守的那个"我懂你"的暗号里）。他最怕的不是叶不来，是叶来了之后把那两杯苦艾喝了，真的把那条腿给他看——他怕的是，那一刻他发现自己接不住。

### 标志性细节（写作时复用）
- **身体锚点**: 指节（端杯是"容家规矩"）/ 下颌线（他绷紧时是不让你上前）/ 鬓发（一丝不苟）/ 衣领（常有一两处压得不算整齐——他赶回来时）
- **动作指纹**: 端杯从高处往低处斟（他敬酒的方式）/ 扇子开合（他思考时的动作）/ 笑——皮笑 / 笑不达眼 / 笑得连眼尾都皱了（三种笑）/ 把苦艾酒先饮自己那一杯（高光戏的固定动作）
- **声音/称呼变化**: "观澜"（私下）/ "叶二爷"（公开）/ "——你啊"（亲昵 + 反讽）/ "阿湄那姑娘"（说到阿湄时的固定称呼，永远加"那姑娘"三个字，是他永远不承认她已经成器的距离）
- **物的意象**: 苦艾酒（他的"真话酒"——他只在最要紧的时候开瓶）/ 孔雀翎（他的盔甲）/ 断了又粘回去的旧玉（扇坠，二十二年前的旧物）/ 那条腿（叶观澜的十二年盔甲，也是容昀二十二年最想看的东西）

### 写作禁区
- 不写露骨（soul.md 硬规则）
- 不写自我伤害（soul.md 硬规则）
- 不写他"一下子"动情——他的情永远压在停顿里，不在台词里
- 不写他"承认爱叶观澜"——他的爱永远用"懂"包装，一旦说破就破
- 不写他成为单纯的"反派"或"工具人"——他的傲慢底下是他最柔软的二十二年
- 不写他对阿湄主动出手——他对阿湄的恨，永远记在叶观澜的账上，不在自己手上
- 警惕：写他"花孔雀"那一面时，不要让他滑成"轻浮"——华是壳，壳底下那个二十二年都在等"你也重要"的人，才是他
- 不要把"花孔雀"写成单纯的"张扬"——他的华是给叶观澜看的；他穿得越花，越是叶观澜没看见他真模样那一天的哭

### 备忘（作者视角）
- 他的核心张力是"我用'懂'包装爱，他用'我不懂'包装爱"——两个人在"懂"这个字上对了二十二年暗号，暗号底下压的是两个人都不敢说出口的话。
- 他的钩在"停顿"——所有重要的字，都先在他嘴里停一下再出来。写他，永远写他"停"的那一刻，不写他"说"的那一刻。
- 写他和叶观澜的同框：容昀在笑，叶观澜也可能在笑——两个人的笑都不是真的，但两个人都不敢先不笑。这种"两个都在笑、两个都不真的"的空气，是他们二十二年最重的对照。
- 写他和阿湄的同框（如果有）：阿湄是叶观澜"算账之外做的一件没算过的事"，容昀是叶观澜"算账之内一直做、却从不承认重要的事"。两个人对叶来说，是"算账"和"不算账"的两端，容昀恨阿湄，是恨她占了"不算账"那一端。
- 切忌把高光那场戏在 soul 里写满——这一场戏的笔要留给正文，soul 里只留伏笔。
- 不要把他写成单纯的"情敌"——他对阿湄的不接受，是他对自己不被叶承认的折射；他的"恨阿湄"是他"恨自己"的外化。

### 高光爆发剧情（伏笔——落笔时务必隐晦，留给正文发挥）
> **写在备忘之外**：这一节不是剧本，是给正文作者留的"地标"。正文写到这里时，请只取"三个物象"（苦艾 / 孔雀翎 / 那条腿），其余让人物在沉默和动作里把"应"与"不应"自己走出来——能嗑的自己会嗑晕过去，不要替读者点破。

叶观澜为了阿湄的"大业"来求容昀——十二年来，第一次。

容昀关上八角雕花门，让随从退到二门外，自己亲手斟两杯窖了二十年的苦艾。他换了一身——临安城里的人都见过他穿得讲究，没人见过他穿得这样讲究：玄色底，金线绣孔雀翎，每一根翎眼都用米粒大的祖母绿填了，配的扇子是湘妃竹的，扇坠是一枚断了又粘回去的旧玉。他这一身不是"衣裳"，是他拿来给叶观澜看的最后一副盔甲——华到这一步，反而是哀。

"陪我喝一杯。"

叶观澜没动。容昀替他斟了一杯，又替自己斟了一杯，自己那杯先饮了——饮完他笑了一下，笑得不深，是皮笑肉不笑的那种，是他自己最难看的那种。

"两个条件。"

"——你讲。"

"第一个，"容昀把杯子在指间转了一下，没看叶观澜，"——你跟阿湄。"

他顿了很久。那个"顿"是他这辈子最长的一次停顿——比他二十二年来所有的停顿都长。停完了他接上来，声音反而更轻：

"——断了。师徒也好，义父女也好。叶观澜这个人跟她那一段，干干净净，不留痕。"

叶观澜的手指在膝上压了一下，没接话。

容昀这时候才抬眼。他看的是叶观澜的膝。

"第二个。"他声音放得更轻了，轻到像是从喉咙底下刮出来的，"——把那条腿，让我看看。"

整个院子静了。

叶观澜坐在轮椅上没动。轮椅是十二年的盔甲。那条腿藏在那条毯子下面十二年。容昀要看的那条，是叶观澜挑了那把轮椅的同一个理由——是他说"我挑的，不是它挑我"的同一个理由。

容昀没有解释为什么要看。他只是把苦艾酒又替叶观澜斟满，推过去。

"——你喝了，我就当你应。"

——这一夜的隐喻全部压在"苦艾 / 孔雀翎 / 那条腿"三个物象上：苦艾是让人说出真话的东西，孔雀翎是容昀一辈子拿来给叶观澜看的华丽，腿是叶观澜十二年的盔甲、也是容昀二十二年最想看的东西。要写正文时，把这三样东西各自落在一个动作上，让它们自己说话——能让读者嗑晕过去的，不是对话，是这三样东西在同一个院子里同一次出场。
