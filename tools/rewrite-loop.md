# 重写循环 · 全章升格到「全球爆款」档

> 目标:把每一章重写成超高质量正文。北极星 = **人真(有血有肉)·世界真(难忘)·爽点足·GoT 多线·全球爆款**。
> 架构:外层循环每轮**只重写一章**,从队列取章 → 交 `bianjibu` swarm 写+审+回炉 → **过门才前进**。
> 不在本 prompt 里手写"怎么写好":那是 `bianjibu` + `docs/standards` 的事。本文件只管**调度 + 卡门 + 状态**。

---

## 〇、北极星 → 五道可勾选的门(每章必过,不达标即回炉)

把"爆款"拆成五个**能验收**的问题。每章重写完,审校必须逐条回答 YES,否则回炉:

| 门 | 北极星 | 验收一问(YES/NO) | 出处 |
|---|---|---|---|
| **G1 人真** | 有血有肉 | 把这章 POV 切到任一配角,他这一章是否过着**有他自己分量的一天**?(有自己的目标/心理戏/独有动作,抽走林夙仍成立) | CLAUDE.md §三 |
| **G2 世界真** | 难忘 | 这章的大环境压力(经济/政治/命册/文过饰非)是被**"感到"**还是被"讲到"?有无一处"对不上账"的细节让世界有重量? | world-macro.md §写法戒律 |
| **G3 爽点足** | 爽 | 这章有没有明确情绪兑现(爽/痛/转)+ 一句能截图转发的话?开场强度 ≥7? | rubric.md |
| **G4 多线/平衡** | GoT | 本章主导频标清(男频/女频/混合)?是否守 §零-2 配额?章末优先「切走式钩子」?POV 未连续 6 章同一人? | CLAUDE.md §零 |
| **G5 双层** | 爆款 | 一笔写两层:空心读者只读表层也爽完,高级读者多赚一层文学红利?深层没写进表层承重墙? | 雅俗共赏.md |

> **数值门(地板,先过再谈五门)**:`engine/prose_lint.py` 干净 + `wenbi-review` ≥ **12/14** + 9 硬门全过。
> 数值是地板,五门是天花板。**只刷到 12 分但 G1/G2 答 NO,照样回炉**——爆款靠的是"难忘",不是分数。

状态机:`TODO → DOING → (过五门?) → DONE | (回炉<3?) → DOING | BLOCKED`

## 二、外层 LOOP PROMPT(喂给 claude-loop / 每轮原样跑)

```
你在执行《镇狱之渊》全章重写循环。本轮只做一章,做完即停。

1. 读 editor_progress.md 的队列表,取**最靠前的 TODO**章(没有 TODO 就报告"全部完成或 BLOCKED"并停)。标它 DOING。
2. 读这一章原文 + 前后各 2 章(连续性) + chronicle/INDEX.md(查最近 15 章 cast 频率、缺席最久的角色、已有锚点)。
3. 调用 bianjibu skill,对这一章执行「写+多路审校+回炉+落地」。把本轮 北极星五门(G1-G5,见 rewrite-loop.md §〇)作为审校额外硬门一并交给它。
4. 过门判定(全满足才算 DONE):
   - python engine/prose_lint.py <章文件> 干净
   - wenbi-review 评分 ≥ 12/14 且 9 硬门全过
   - G1-G5 审校逐条 YES
   不满足 → 回炉(回炉数+1)。回炉数已达 3 仍不过 → 标 BLOCKED,写清卡在哪门,跳过。
5. 过门 → 落 frontmatter + 正文,标 DONE,在 editor_progress.md 记一行(数值门/五门/回炉数/备注)。
6. 本地构建 + tools/publish.sh 门禁验证;过了再 git 提交这一章(commit: "rewrite: ch### 升格爆款档")。不 push(等用户批量 push)。
7. 停。等下一轮。

铁律:一轮一章。绝不一次改多章。绝不跳过过门直接落地。绝不在一章上回炉超 3 次。
```

## 三、为什么这样切(给人看的设计说明)

- **外层廉价、内核冷启**:外层只做"取章/判门/记状态",省 context;真正费 token 的写作交 `bianjibu`,它内部 spawn 策划/写手/审校子 agent,**每个冷启动 = context 干净**,质量不随循环衰减。
- **五门防"刷分陷阱"**:纯数值门会让 AI 学会写"高分但忘不掉个屁"的安全章。G1/G2 强制回答"人能不能独立成立""世界是不是被感到",把"难忘"变成硬门。
- **回炉有上限**:爆款不是把一章磨到完美,是**整本均质地难忘**。一章卡 3 轮就 BLOCKED 留人工,让循环往前走,避免在单章上烧光预算。
- **GoT 多线靠 G4 + 跨章**:单章过 G4 只保证本章平衡;**配额(每 10 章男频≥3/女频≥3)、POV 轮换、线不缺席>8 章**这些跨章规则,由队列表的"备注"列累积追踪——每 10 章回看一次配额,跑偏就把后续 TODO 章的主导频调过来。

## 四、跑法

- **手动逐轮**:把 §二 prompt 贴进会话,跑一轮看一章,满意再继续。适合头几章校准门的松紧。

- **校准期建议**:先手动跑 ch1-3,确认五门松紧合适(太松出不了爆款,太紧全 BLOCKED),再放自动。
