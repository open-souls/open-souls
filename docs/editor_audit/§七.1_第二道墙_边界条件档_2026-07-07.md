## 二、撞墙病根（主编独立诊断 + Loop #91 §四.2 已记录）

### 病因
**写"信息差 / 布局视角" + 写"角色心理戏" = 天然需要"他做这一手是为了什么" = 一旦解释就掉自我指涉坑**

### 妥协结果
为了不违规"直接写心理活动"，AI 只能用「动的来处是山顶」「按住的方式是不承认的按住」这种**伪意象、真解释**的句式蒙混过关——把"只写感知，不要下判断"的规则**机械理解成"把所有心理活动用 X 的来处是 Y、X 的方式不是 X 公式化翻译"**。

### 灾难以降级为"读不懂"
- ch791 一段 200 字内"来处"出现 8 次
- ch792 出现"按住按住按住按住按住按住按住"机械复读
- ch793 几乎**每一句都是"X 的方式不是 X / X 的来处是 Y"**的嵌套
- 小说彻底失去文学美感，变成干瘪的逻辑链条

## 三、错误示范 vs 正确改法（直接复用 Loop #91 主编范例）

### ❌ 错误示范
> "她按住刀柄的方式是她自己不肯承认的那种握。"
> "动的来处是山顶。山顶的来处是叶观澜。叶观澜今天第一次让山顶比过去六十年轻的那一种让。"
> "焐的方式不是焐——是她自己不肯承认的那种焐——焐的来处是山顶。"

### ✅ 正确改法
> "她手搭在刀柄上，指节在袖里焐得有些发白，再没有松开。"

### 核心原则
**留白的来处是物理动作和环境的沉默，而不是用字面去解释"这叫不承认"。**

- ❌ 不要把"她不肯承认"翻译成"她按住的方式是她自己不肯承认的那种按住"
- ✅ 用身体/物象/环境来承担——指节发白 / 焐得发白 / 没有松开 = 读者自己读出"不肯承认"

## 四、撞墙的 POV 类型（边界条件）

| POV | 撞墙深度 | 原因 |
|---|---|---|
| **林夙 POV（被压后自我反刍）** | 中 | "我看见 X" + "X 是什么" = 双向掉坑 |
