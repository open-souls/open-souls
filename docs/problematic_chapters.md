### 1.3 病根（不是破折号本身）

**所有高 em-dash 章都是同一个病根**：「X 的来处是 Y / X 的方式不是 X / X 的位是 Y / X 的方向是 Y」四种机械翻译公式的连环嵌套。

这是 §七.1 第二道墙——`docs/editor_audit/§七.1_第二道墙_边界条件档_2026-07-07.md` 已记录，Loop #91 实测"主编亲自写也破不了"。

**病体样本（ch826-还有气.md，第 27-29 行）**：
```
知道的方式不是知道——是苏挽今天第一次帮阿湄托住林夙的肩——托住肩的那一拍她自己手下的气还在——还在的方式不是还在——是苏挽今天第一次帮阿湄托住林夙的肩——托住肩的那一拍她自己知道林夙还有一线气。
```

3 句话里 **5 个破折号 + 6 个"X 的方式不是 X / X 的来处是 Y"**。这是 subagent 把"只写感知不写心理"机械翻译的产物。

## 2. WARN 级别 · 真实数字

### 2.1 拉丁字母残留

原报告：6 章残留。
实测：**232 章** (66.1%) 有残留——因为几乎所有 chapter 都用 `score: 14/14` 或 `LOW` 之类英文 token。

**但这些不是文笔污染**——它们是 **frontmatter YAML 字段名**（`chapter: 826` / `pov: 阿湄` / `score: 14/14` / `hook:` / `LOW` / `vs` / `FINAL` / `II` / `cast:`），不是正文里的英文残留。

| 类型 | 数量 | 是否需修 |
|------|------|---------|
| 纯 frontmatter 字段名 | ~210 章 | **否**（YAML 必备，删除会破坏 frontmatter 解析） |
| frontmatter 内的状态标签（`LOW` / `FINAL` / `II`） | ~30 章 | 是（应改中文） |
| 正文中英文残留 | 0 章 | 否（实测 = 0） |

**真正需修的是 frontmatter 内的 `LOW` / `FINAL` / `II` 标签**（约 30 章）。这些是 subagent 评分/阶段的英文记号，正文化即可。

### 2.2 碎片率偏高（原报告 35 章）

原报告 1-3 字短句占 30-41% 触发。
**实测发现**：ch826 这种 em-dash 链病体本身就是"碎片率 100%"——因为每半句话一个破折号断开。所有 315 章 ERROR 都同时碎片率超标。

所以**碎片率 = em-dash 的子集，不需单独统计**。**修 em-dash 自动修碎片率**。
