# 综合分析 · rubric 14 分 + 节奏硬门

## 0. 范围与方法

**数据源**：10 份 RANGE 报告（RANGE-01..10），每份覆盖 50 章、抽样 20 章（每 5 章抽 2），全书 502 章共 200 章样本。

**rubric 7 项定义**（来自 `docs/standards/rubric.md`，每项 0/1/2）：
- **钩子**：结尾有没有让人想追下一回的悬
- **爽痛**：这回有没有明确的情绪兑现（爽/痛/转）
- **反差**：有没有用到某角色的"说 vs 做 / 表 vs 里 / 被逼到墙角"
- **拉扯**：关系靠推拉前进，而不是把张力解决掉
- **记忆点**：有没有一句能被截图/转发的话，或一个强画面
- **代入**：有没有一个清晰的视角让读者钻进去
- **新**：桥段是新用/混搭，还是套路平铺

总分 0-14，**< 9 退回重写一次**。

**节奏硬门**（不计分；不达标退回重写一次）：
- 开场强度 ≥ 7/10
- 转折卡 22% / 47% / 68% 字数
- 89% 处再翻一次 + 留钩

**聚合口径**：
- RANGE-01/03/07/08/10（ch11-60/111-160/311-360/361-410/461-502）：直接用其 rubric 14 分
- RANGE-02（ch61-110）：6 维 5 分制打分（文笔/人物/frontmatter/编剧/母题/综合），推算 rubric ≈ 综合×14/5×0.95（"综合"已含钩/爽痛意涵）；仅作趋势参考，不计入 < 9 退回判定
- RANGE-04（ch161-210）/ RANGE-05（ch211-260）/ RANGE-06（ch261-310）/ RANGE-09（ch411-460）：6 维 5 分制，**未直接给 rubric 14 分**，仅给出"6 维均分/5"；按"6 维均分×14/5×0.95"换算为参考性 rubric（标记 ⚠），**不参与 < 9 退回判定**
- 本表" < 9 退回率"仅统计有 rubric 14 分直接打分的范围（RANGE-01/03/07/08/10）

## 2. rubric 七项的趋势

按段（R01→R03→R07→R08→R10）观察：

- **钩子**：1.85 → **2.00（峰值）** → 1.75 → 1.80 → **1.40（谷底）**。前段稳定在 1.85-2.00 区间，末段断崖至 1.40。**末段章末钩退化为"未说出的后半句链"**，被 R10 报告点名批评。
- **爽痛**：1.50 → 1.95 → 1.65 → **1.30** → **0.95**。最差的一项，下降曲线最陡。**R08 报告注："所有章都用解释性补叙"** 是爽痛不能落地的根本原因。
- **反差**：1.95 → 2.00 → 1.85 → **1.30** → 1.25。**R08 全 1 分**——林夙 POV 全场无反差可言。
- **拉扯**：1.50 → 1.80 → 1.85 → 1.40 → 1.40。前段上行、末段下行。R01/R03 都是"拉扯中等"，因为 50 章内对手戏场次限制；末段 ch411-502 多为集合/集结/收网戏，少二人推拉。
- **记忆点**：1.95 → 2.00 → 1.90 → 1.55 → 1.65。前段顶级，末段塌方但比"爽痛/反差"略好（ch502、ch396 仍有范文级记忆点）。
- **代入**：2.00 → 1.95 → 1.85 → **1.05（崩溃）** → 1.40。R08 代入 1.05 是全表最低均值——林夙 POV 全锁，读者无替代视角钻入。
- **新**：1.10 → 1.85 → 1.20 → 0.95 → **0.75（崩溃）**。唯一全程走低的项。"新"是 rubrick 七项中最稀缺的稀缺资源：R01 起就 1.10（"本段是节奏复刻期"），R10 末段降至 0.75（"桥段模板复用"）。

**总结**：
- **稳定强项**：钩子、记忆点（R01-R07 均稳定 1.85+）
- **末段断崖项**：爽痛、反差、代入、新（R07→R10 全部掉 0.5+）
- **最早弱项**："新"——R01 就 1.10，是全书的慢性病
- **末段最崩溃项**：代入（R08 1.05）、新（R10 0.75）
