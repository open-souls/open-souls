# Review Protocol · 主编审稿 9 条硬门

> 主编 review agent 输出时，逐章过这 9 条。任一不过，整批打回。

## 0. Pre-flight

```bash
# 范围确认
ls seasons/01-xianxia/chronicle/ | grep -E "^(NNN-)" | head
# Lint baseline
PYTHONIOENCODING=utf-8 python engine/prose_lint.py
```

过线条件：**0 ERROR**。

---

## 1. 硬线 (HARD GATE)

| 检查 | 工具 |
|------|------|
| 无露骨 | `tools/review_batch.py` 内 HARDLINE_PATTERNS |
| 无自伤 | 同上 |
| 林窈 (13) 不涉及暧昧 | 同上 |

任意硬线违例 = **整批打回**，不进入下游。

---

## 2. 锚点 (Anchor System)

| 等级 | 内容 | 章节距离 |
|------|------|----------|
| 0 级 | 感官锚点建立（无情绪标记） | 起点 |
| 1 级 | 距离缩短 + 未竟停顿 | 0→1 之间 ≥3 章 |
| 2 级 | 主动小动作 | 1→2 之间 ≥3 章 |
| ... 跳级禁止 | 相邻差 ≤1 | ... |
| 6 级 | 发生 | 仅 CP 完成 |

**硬规则**: 不许一章连跳两级。每次 review 时数"该 CP 自上一章过了几级"。

---

## 3. 前情钩 (Hook)

**两种钩，必居其一**：

| 类型 | 用法 | 例 |
|------|------|-----|
| 画面钩 | 章末留下一个未完成的视觉/动作 | "她往门口走，没回头——林夙知道，她不是去接林崇的。" |
| 信息钩 | 章末留下一个未揭的谜/转折 | "叶观澜知道林崇要来——比林崇决定来，还早三日。" |

**禁止**:
- 总结式结尾（"这就是叶观澜的局"）
- 议论式结尾（"那一刻林夙终于明白……"）
- 双钩并存（最后一章一钩）

**frontmatter hook 字段含引号时必须 `|` block-scalar**（YAML 解析要求）。

---

## 4. 缺席上限 (Cast Rotation)

- 任何配角不得连续缺席 > 10 章
- 阿湄专项：不得连续 3 章无她
- 新角色破场后不得连续 > 5 章（防喧宾夺主）
- **不得连续 4 章 cast 数量相同**（节奏硬规则）

每章写前必须统计最近 15 章 cast 频率。

---

## 5. 伏笔回改 (Refactor Policy)

**允许回改**:
- 现有场景加 1-3 句
- 不影响剧情推进方向
- 改完在 `ships` 字段标 `【已回改·锚点说明】`
- git commit 注明 `refactor: 回改 ch### 埋伏笔`

**禁止**:
- 改对话主干
- 删已有锚点
- 二次回改已被标记的章节

---

## 6. 节奏 (Beat)

每章 frontmatter `beat` 字段必须含：
- **弧内章**：承 / 转 / 合 + 当前 in_beat / 总数 + 章节功能描述
- **弧起始章**：起 + 弧描述 + in_arc（如 `起·噩耗传开，阿湄压制崩裂，苏挽向内收 1/12`）

```yaml
# 弧内章
beat: 转·宗门内部暗流，轮回秘密险些全揭 31/31
# 弧起始章
beat: 起·噩耗传开，阿湄压制崩裂，苏挽向内收 1/12
```

**起 用法**：仅用于跨 arc 的开场章（如 ch001、ch066、ch401）。一旦转入弧内，下一章必须用 承/转/合。

---

## 7. 声音 (Voice)

| 角色 | 语速 | 关键词 |
|------|------|--------|
| 林夙 | 极轻、平、停一下 | "嗯", "他道，声音轻，平" |
| 阿湄 | 随意、压着、跳 | "她道，声音随意" |
| 苏挽 | 客气、疏离、不取回 | "她道，声音客气而疏离" |
| 裴无咎 | 痞、懒、停一停 | "他道，声音痞" |
| 牛阿大 | 大、直、快 | "他道，声音直" |
| 余伯 | 不说快话、低 | "他道，声音低" |

**禁止**:
- 林夙说快话
- 苏挽说软话
- 裴无咎说沉话
- 三人一起说大道理

---

## 8. 群像 (Ensemble)

每个出场角色必须有至少一件**只属于自己的事**:
- 独有的小动作
- 体现声线的台词
- 只有他/她才有的情绪反应

**禁止工具人式出场**: 角色不是为了给林夙递消息。

---

## 9. frontmatter 标准

```yaml
---
season: N
chapter: NNN
title: 2-4字标题
cast: [角色A, 角色B, ...]  # 最少 4 人（节奏硬规则）
pov: POV 角色
beat: 承/转/合·描述 N/M
ships:
  CP×CP: 本章埋的伏笔或情感张力
hook: 章末钩子（用 | 当含引号）
---
```
